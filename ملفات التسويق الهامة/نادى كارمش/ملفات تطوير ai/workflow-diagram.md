## 🎯 تفاصيل كل Node

### 1️⃣ **Schedule Trigger** ⏰
```yaml
النوع: Cron Schedule
التكرار: كل 8 ساعات
الأوقات: 00:00 | 08:00 | 16:00
التكلفة: مجاني
```

### 2️⃣ **Google News RSS** 📰
```yaml
النوع: HTTP Request
الرابط: news.google.com/rss
البحث: "محرم بك الإسكندرية"
اللغة: عربي
النتائج: ~10-20 خبر
التكلفة: مجاني تماماً
```

### 3️⃣ **GNews API** 🌐
```yaml
النوع: HTTP Request
الرابط: gnews.io/api/v4
Free Tier: 100 طلب/يوم
النتائج: 10 أخبار/طلب
التكلفة: مجاني (يكفي 3 تنفيذات/يوم)
```

### 4️⃣ **Merge & Filter** 🔀
```javascript
الوظيفة: JavaScript Code Node
المهام:
  - دمج من مصدرين
  - إزالة المكررات
  - فلترة حسب الجودة
  - ترتيب حسب التاريخ
  - اختيار أفضل 5

الخوارزمية:
  1. جمع كل الأخبار
  2. filter(unique by title)
  3. sort(by publishedAt DESC)
  4. slice(0, 5)
```

### 5️⃣ **Qwen AI Analysis** 🤖
```yaml
النوع: HTTP Request to Groq API
Model: qwen2.5-7b-instruct
السرعة: ~2-5 ثواني/طلب
الجودة: ممتازة للعربية 🇪🇬
التكلفة: مجاني 100% (30 req/min)

Prompt Template:
  System: "أنت محلل محتوى لمحرم بك..."
  User: "الخبر: {title}..."
  Output Format: JSON
```

### 6️⃣ **Parse AI Response** 📦
```javascript
المهام:
  ✓ استخراج JSON من النص
  ✓ معالجة markdown code blocks
  ✓ Error handling
  ✓ دمج مع البيانات الأصلية

Output Structure:
{
  title: "...",
  url: "...",
  ai_analysis: {
    summary: "...",
    posts: [...],
    hashtags: [...],
    image_suggestion: "..."
  }
}
```

### 7️⃣ **Generate Image** 🎨
```yaml
API: Together AI
Model: FLUX.1-schnell
Size: 1080x1080px
Steps: 4 (سريع)
Quality: High
التكلفة: $0.002/صورة

Prompt Construction:
  "Egyptian city scene, 
   {image_suggestion}, 
   Moharram Bek Alexandria,
   modern urban photography,
   high quality, vibrant colors"
```

### 8️⃣ **Prepare Posts** 📝
```javascript
التحويل: خبر واحد → 3 بوستات

Loop through ai_analysis.posts:
  - Post 1 (رسمي)
  - Post 2 (شبابي)  
  - Post 3 (تفاعلي)

كل بوست يحتوي على:
  ✓ النص الكامل
  ✓ الهاشتاجات
  ✓ الصورة
  ✓ الأسلوب
  ✓ المصدر
```

### 9️⃣ **Save to Notion** 💾
```yaml
النوع: Notion API Integration
العملية: Create Database Page

الحقول:
  - العنوان (Title)
  - نص البوست (Rich Text)
  - الأسلوب (Select)
  - الهاشتاجات (Rich Text)
  - رابط الصورة (URL)
  - المصدر (Rich Text)
  - رابط الخبر (URL)
  - الحالة (Select: "للمراجعة")
  - التاريخ (Date)
```

### 🔟 **Telegram Notification** 📲
```yaml
النوع: Telegram Bot API
التنسيق: Markdown

Message Template:
  🔔 بوست جديد جاهز!
  📰 الخبر: {title}
  ✍️ الأسلوب: {style}
  📝 النص: {text}
  🏷️ الهاشتاجات: {hashtags}
  📷 الصورة: {url}
  👉 رابط Notion للمراجعة
```
