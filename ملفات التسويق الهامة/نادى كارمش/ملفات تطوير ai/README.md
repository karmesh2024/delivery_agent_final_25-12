## 📖 نظرة عامة

هذا النظام يقوم بـ:
1. **جمع** الأخبار من مصادر متعددة (Google News, GNews API)
2. **تحليل** المحتوى باستخدام Qwen 2.5 (AI عربي متقدم)
3. **توليد** 3 أنماط مختلفة من البوستات (رسمي، شبابي، تفاعلي)
4. **تصميم** صور مخصصة باستخدام AI (Flux)
5. **حفظ** كل شيء في Notion للمراجعة البشرية
6. **إشعار** فوري عبر Telegram

كل هذا **تلقائياً** كل 8 ساعات! ⏰

## 🚀 التنصيب السريع

### المتطلبات الأساسية
- [ ] Docker مثبت
- [ ] حساب Groq (مجاني)
- [ ] حساب Together AI (مجاني)
- [ ] حساب Notion (مجاني)
- [ ] حساب Telegram
- [ ] حساب GNews (مجاني)

### الخطوات (5 دقائق)

#### 1. تشغيل n8n
```bash
docker run -d \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  docker.n8n.io/n8nio/n8n
```

#### 2. افتح n8n
```
http://localhost:5678
```

#### 3. استيراد الـ Workflow
- Workflows → Import from File
- اختر `moharram-bey-content-automation.json`

#### 4. إضافة API Keys
- اتبع التعليمات في `SETUP-GUIDE.md`
- أضف Credentials لكل service

#### 5. تفعيل
- اضغط **Active**
- انتهى! ✅

**📚 للتعليمات المفصلة:** راجع `SETUP-GUIDE.md`

## 🔍 Workflow Details

### Nodes المستخدمة
| Node | الوظيفة | الوقت |
|------|---------|-------|
| Schedule Trigger | التشغيل التلقائي | - |
| Google News RSS | جمع أخبار | 5s |
| GNews API | جمع أخبار | 3s |
| Merge & Filter | دمج وفلترة | 1s |
| Qwen Analysis | تحليل AI | 15s |
| Parse Response | استخراج JSON | 1s |
| Generate Image | صورة AI | 25s |
| Prepare Posts | إعداد البوستات | 2s |
| Save to Notion | حفظ | 5s |
| Send Telegram | إشعار | 2s |

**الإجمالي:** ~60 ثانية لكل خبر × 5 = ~5 دقائق

## 🆘 حل المشاكل

### Problem: Workflow لا يعمل
```bash
# تحقق من Logs
docker logs n8n

# أعد التشغيل
docker restart n8n
```

### Problem: Authentication Failed
```
✓ تأكد من API Keys صحيحة
✓ تحقق من Bearer Token format
✓ جدد المفتاح إذا انتهى
```

### Problem: Rate Limit
```
• Groq: 30 req/min (انتظر دقيقة)
• GNews: 100 req/day (قلل التنفيذات)
• Together AI: تحقق من Credits
```
