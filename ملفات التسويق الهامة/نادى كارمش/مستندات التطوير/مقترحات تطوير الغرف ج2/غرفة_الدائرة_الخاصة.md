<a name="concept"></a>
## 1️⃣ المفهوم الأساسي

### ما هي غرفة الدائرة؟

> **"غرفة الدائرة هي المساحة الخاصة والآمنة التي يجتمع فيها أعضاء الدائرة فقط"**

```
الفرق الجوهري:
━━━━━━━━━━━━━━

الغرف العامة (🏛️):
├─ مفتوحة للجميع
├─ محتوى عام
├─ تعارف وبناء علاقات
└─ هدف: جمع البيانات + اكتشاف

غرف الدوائر (🏠):
├─ خاصة بالأعضاء فقط
├─ محتوى حصري
├─ تعميق العلاقات
└─ هدف: التواصل العميق + التعاون
```

### الهدف من غرفة الدائرة

```
الأهداف الأربعة:
━━━━━━━━━━━━━━━

1️⃣ التواصل الخاص:
   محادثات خاصة بين الأعضاء
   
2️⃣ مشاركة الموارد:
   كتب، ملفات، روابط مفيدة
   
3️⃣ التخطيط المشترك:
   تنظيم لقاءات، مشاريع، أحداث
   
4️⃣ بناء العمق:
   علاقات أعمق من مجرد "متابعين"
```

## 2️⃣ الفرق بين الغرف العامة وغرف الدوائر

### جدول المقارنة الشامل

| الميزة | الغرف العامة 🏛️ | غرف الدوائر 🏠 |
|--------|-----------------|----------------|
| **الوصول** | الجميع | الأعضاء فقط |
| **الخصوصية** | عامة | خاصة جداً |
| **المحتوى** | منشورات عامة | محادثات خاصة |
| **الهدف** | اكتشاف وتعارف | تعميق وتعاون |
| **عدد الأعضاء** | غير محدود | محدود (3-15) |
| **الإدارة** | Admins | منشئ الدائرة |
| **الميزات** | Posts, Comments | Chat, Files, Events |
| **البيانات** | للـ Profiling | خاصة 100% |

## 4️⃣ الميزات والوظائف

```

- خالد (مسوق)

الميزات النشطة:
```

```typescript
export class CircleRoomsController {
  
  /**
   * GET /api/circles/:circleId/room
   * جلب غرفة الدائرة
   */
  async getCircleRoom(req: Request, res: Response) {
    try {
      const { circleId } = req.params;
      const userId = req.user.id;
      
      // التحقق من العضوية
      const member = await prisma.circleMembers.findUnique({
        where: {
          circleId_customerId: { circleId, customerId: userId }
        }
      });
      
      if (!member) {
        return res.status(403).json({
          success: false,
          error: 'غير مصرح لك بالدخول'
        });
      }
      
      // جلب الغرفة
      let room = await prisma.circleRooms.findFirst({
        where: { circleId }
      });
      
      // إنشاء الغرفة إذا لم تكن موجودة
      if (!room) {
        room = await prisma.circleRooms.create({
          data: { circleId }
        });
      }
      
      // جلب البيانات الإضافية
      const [messages, files, events, members] = await Promise.all([
        this.getRecentMessages(room.id, 50),
        this.getRoomFiles(room.id),
        this.getUpcomingEvents(room.id),
        this.getRoomMembers(circleId)
      ]);
      
      return res.json({
        success: true,
        data: {
          room,
          messages,
          files,
          events,
          members
        }
      });
      
    } catch (error) {
      console.error('Get circle room error:', error);
      return res.status(500).json({
        success: false,
        error: 'فشل جلب غرفة الدائرة'
      });
    }
  }
  
  /**
   * POST /api/circles/:circleId/room/messages
   * إرسال رسالة
   */
  async sendMessage(req: Request, res: Response) {
    try {
      const { circleId } = req.params;
      const userId = req.user.id;
      const { content, attachments, mentions } = req.body;
      
      // جلب الغرفة
      const room = await prisma.circleRooms.findFirst({
        where: { circleId }
      });
      
      if (!room) {
        return res.status(404).json({
          success: false,
          error: 'الغرفة غير موجودة'
        });
      }
      
      // إنشاء الرسالة
      const message = await prisma.circleMessages.create({
        data: {
          roomId: room.id,
          senderId: userId,
          messageType: attachments?.length > 0 ? 'file' : 'text',
          messageContent: content,
          attachments: attachments || [],
          mentions: mentions || []
        }
      });
      
      // تحديث إحصائيات الغرفة
      await prisma.circleRooms.update({
        where: { id: room.id },
        data: {
          totalMessages: { increment: 1 },
          lastActivityAt: new Date()
        }
      });
      
      // إرسال عبر WebSocket للأعضاء النشطين
      this.broadcastMessage(circleId, message);
      
      // إشعارات للمذكورين
      if (mentions?.length > 0) {
        await this.sendMentionNotifications(mentions, message);
      }
      
      return res.json({
        success: true,
        data: message
      });
      
    } catch (error) {
      console.error('Send message error:', error);
      return res.status(500).json({
        success: false,
        error: 'فشل إرسال الرسالة'
      });
    }
  }
  
  /**
   * POST /api/circles/:circleId/room/files
   * رفع ملف
   */
  async uploadFile(req: Request, res: Response) {
    try {
      const { circleId } = req.params;
      const userId = req.user.id;
      const file = req.file; // من multer
      
      if (!file) {
        return res.status(400).json({
          success: false,
          error: 'لم يتم رفع ملف'
        });
      }
      
      // رفع للـ S3 أو Storage
      const fileUrl = await this.uploadToStorage(file);
      
      // جلب الغرفة
      const room = await prisma.circleRooms.findFirst({
        where: { circleId }
      });
      
      // حفظ معلومات الملف
      const circleFile = await prisma.circleFiles.create({
        data: {
          roomId: room.id,
          uploadedBy: userId,
          fileName: file.originalname,
          fileType: file.mimetype,
          fileSize: file.size,
          fileUrl,
          category: this.detectFileCategory(file.mimetype)
        }
      });
      
      // تحديث إحصائيات
      await prisma.circleRooms.update({
        where: { id: room.id },
        data: {
          totalFiles: { increment: 1 }
        }
      });
      
      // إشعار الأعضاء
      await this.notifyFileUpload(circleId, circleFile);
      
      return res.json({
        success: true,
        data: circleFile
      });
      
    } catch (error) {
      console.error('Upload file error:', error);
      return res.status(500).json({
        success: false,
        error: 'فشل رفع الملف'
      });
    }
  }
```
