## 🎯 قواعد عامة لكل الغرف

### متى تُطرح الأسئلة:

```yaml
أسئلة الدخول:
  - تُطرح في أول 3 زيارات فقط
  - بعد ذلك: دخول مباشر
  - إلا إذا تغيرت الاهتمامات

أسئلة الخروج:
  - تُطرح بعد قضاء 5+ دقائق في الغرفة
  - تُطرح مرة كل 5 زيارات (ليس كل مرة)
  - يمكن تخطيها دائماً

أسئلة إضافية أثناء التصفح:
  - بعد أول بوست (سؤال عن الموضوع)
  - بعد 10 تفاعلات (سؤال عن تفضيلات المحتوى)
  - بعد شهر (تحديث الاهتمامات)
```

### مكافآت الإجابة:

```
- إجابة سؤال دخول: 2 نقطة
- إجابة أسئلة متابعة: 1 نقطة لكل سؤال
- إجابة سؤال خروج: 1 نقطة
- إكمال كل الأسئلة في غرفة: 5 نقاط bonus

→ يشجع المستخدمين على الإجابة
```

## ✅ الخلاصة

### ما حققناه:

```
✅ نظام أسئلة ديناميكي لكل الـ 8 غرف
✅ أسئلة دخول وخروج مخصصة
✅ أسئلة متابعة حسب الاختيار
✅ بناء ملف نفسي تدريجي
✅ مكافآت لتشجيع الإجابة
✅ احترام خصوصية المستخدم (يمكن التخطي)
✅ ذكاء في التوقيت (لا إزعاج)
```

### الفوائد:

```
للمستخدم:
- تجربة شخصية
- محتوى مخصص
- مطابقات دقيقة
- لا استبيانات طويلة

للنظام:
- ملف عميق لكل مستخدم
- فهم الأبعاد النفسية
- تحسين المطابقة باستمرار
- بيانات قيمة للتحليل
```
