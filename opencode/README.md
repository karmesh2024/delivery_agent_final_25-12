# لوحة تحكم وكلاء التوصيل - Delivery Agent Dashboard

## نظرة عامة على المشروع

نظام إدارة شامل متكامل مبني على Next.js 15 مع TypeScript، يهدف إلى إدارة جميع جوانب عمليات التوصيل والخدمات المرتبطة بها. يتضمن النظام لوحة تحكم إدارية متقدمة، نظام إدارة المخلفات، بورصة، نظام نقاط، راديو، وإدارة الموردين والمخازن.

## التقنيات المستخدمة

### Frontend
- **Next.js 15.5.7** - إطار عمل React مع App Router
- **React 18.3.1** - مكتبة React
- **TypeScript** - لبرمجة قوية
- **Tailwind CSS** - للتنسيق
- **shadcn/ui** - مكتبة مكونات UI قائمة على Radix UI
- **Redux Toolkit** - لإدارة الحالة
- **React Hook Form** - لإدارة النماذج

### Backend & Database
- **Prisma ORM** - لإدارة قاعدة البيانات
- **Supabase** - قاعدة بيانات PostgreSQL مع خدمات Auth و Realtime
- **Zod** - للتحقق من البيانات

### Mapping & Charts
- **Leaflet & React-Leaflet** - للخرائط
- **Chart.js & Recharts** - للرسوم البيانية
- **React-Leaflet-Draw** - للرسم على الخرائط

### Additional Libraries
- **Framer Motion** - للتحريكات
- **Lucide React** - للأيقونات
- **QRCode** - لتوليد QR codes
- **Axios** - للطلبات HTTP
- **React Hot Toast** - للإشعارات
- **Next Themes** - للوضع الليلي
- **React DnD** - للسحب والإفلات

## الميزات الرئيسية

### 1. نظام إدارة الوكلاء (Agents)
- تسجيل وتفعيل الوكلاء
- تتبع الطلبات والرحلات
- إدارة المواقع والخرائط
- نظام الموافقات

### 2. إدارة الطلبات (Orders)
- إنشاء وإدارة الطلبات
- تتبع الطلبات في الوقت الفعلي
- نظام إشعارات متقدم
- تقارير وتحليلات

### 3. نظام البورصة والمخلفات (Waste Exchange)
- بورصة مخلفات احترافية
- إدارة الفئات والتصنيفات
- نظام تسعير ديناميكي
- تتبع المخلفات وتسعيرها

### 4. نادي Scope Zone والنقاط (Club Zone & Points)
- نظام عضويات متعدد المستويات
- نظام نقاط شامل
- تحويل النقاط بين الأنواع المختلفة
- نظام المكافآت والولاء

### 5. نظام الراديو (Radio)
- بث مباشر للمحتوى
- تتبع المستمعين
- نظام Real-time
- Cron Jobs للبث التلقائي

### 6. إدارة الموردين والمخازن (Suppliers & Warehouses)
- إدارة الموردين الصناعيين
- إدارة المخازن
- ربط المخازن بالمتاجر الإلكترونية
- تتبع المخزون

### 7. الإدارة المالية (Financial Management)
- إدارة المدفوعات
- تقارير مالية
- نظام المحافظ
- تتبع النقاط النقدية

### 8. نظام الصلاحيات (Permissions)
- نظام صلاحيات متعدد المستويات
- نظام الموافقات
- إدارة الأدوار (Admin, Agent, Customer)
- سجل الأنشطة

## هيكل المشروع

```
src/
├── app/                    # Next.js App Router
│   ├── admin/              # لوحة تحكم الإدارة
│   ├── agents/             # إدارة الوكلاء
│   ├── orders/             # إدارة الطلبات
│   ├── waste-management/   # إدارة المخلفات
│   ├── club-zone/          # نادي Scope Zone
│   └── ...                 # صفحات أخرى
├── components/             # المكونات القابلة لإعادة الاستخدام
├── domains/                # منطقة الأعمال
├── infrastructure/         # البنية التحتية
├── services/              # الخدمات الخارجية
├── store/                 # Redux Store
├── types/                 # TypeScript Types
└── utils/                 # أدوات مساعدة

docs/                      # الوثائق باللغة العربية
prisma/                    # قاعدة البيانات
public/                    # الملفات الثابتة
supabase/                  # إعدادات Supabase
```

## الأوامر المتاحة

```bash
# تشغيل بيئة التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build

# تشغيل المشروع في وضع الإنتاج
npm start

# فحص الكود
npm run lint

# تطبيق الترحيلات (Migrations)
npm run migrate

# توليد Prisma Client
npm run postinstall
```


### خطوات التثبيت

1. استنساخ المشروع
```bash
git clone <repository-url>
cd delivery_agent_final_25-12
```

2. تثبيت الحزم
```bash
npm install
```

3. إعداد قاعدة البيانات
```bash
npx prisma generate
npx prisma db push
```

4. تشغيل المشروع
```bash
npm run dev
```
