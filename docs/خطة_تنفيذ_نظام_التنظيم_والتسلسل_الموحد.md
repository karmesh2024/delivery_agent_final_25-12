# خطة تنفيذ نظام التنظيم والتسلسل الموحد

## 📋 نظرة عامة

هذا المستند يوضح خطة تنفيذ نظام موحد لإدارة الفئات والتصنيفات يسمى **"إدارة التنظيم والتسلسل"** يجمع بين:
- نظام التصنيفات الهرمية (القطاعات، التصنيفات، الفئات الأساسية، الفئات الفرعية)
- نظام فئات الكتالوج (الفئات الأساسية والفرعية للمنتجات والمخلفات)
- إدارة الوحدات والبراندز لكل تصنيف

---

## 🎯 الأهداف

1. **توحيد النظام**: نظام واحد لإدارة جميع الفئات والتصنيفات
2. **ربط واضح**: كل فئة مرتبطة بقطاع وتصنيف محدد
3. **مرونة**: إمكانية إضافة وحدات وبراندز لكل تصنيف
4. **سهولة الاستخدام**: واجهة واحدة موحدة
5. **قابلية التوسع**: سهولة إضافة مستويات جديدة

---

## 📐 الهيكل المقترح

```
القطاع (Sector)
  └── التصنيف (Classification)
      ├── الوحدات المتاحة (Available Units)
      ├── البراندز المتاحة (Available Brands)
      └── الفئة الأساسية (Main Category)
          └── الفئة الفرعية (Sub Category)
```

---

## 🗂️ الجداول الجديدة

### 1. `unified_classifications`
- التصنيفات الموحدة
- مرتبطة بـ `warehouse_sectors`
- تحتوي على `item_type` (product/waste/both)

### 2. `unified_main_categories`
- الفئات الأساسية الموحدة
- مرتبطة بـ `unified_classifications`
- تدعم الهيكل الهرمي (parent_id, level, path)

### 3. `unified_sub_categories`
- الفئات الفرعية الموحدة
- مرتبطة بـ `unified_main_categories`
- تدعم الهيكل الهرمي

### 4. `unified_brands`
- البراندز الموحدة
- تجمع `store_brands` و `warehouse_product_brands`

### 5. `classification_units`
- ربط الوحدات بالتصنيفات
- تحديد الوحدات المتاحة لكل تصنيف

### 6. `classification_brands`
- ربط البراندز بالتصنيفات
- تحديد البراندز المتاحة لكل تصنيف

## 🎨 مثال على الكود

### Service Example:

```typescript
// unifiedCategoriesService.ts
export const unifiedCategoriesService = {
  // جلب جميع القطاعات
  async getSectors() {
    const { data, error } = await supabase
      .from('warehouse_sectors')
      .select('*')
      .eq('is_active', true)
      .order('name');
    
    if (error) throw error;
    return data;
  },

  // جلب التصنيفات حسب القطاع
  async getClassificationsBySector(sectorId: string) {
    const { data, error } = await supabase
      .from('unified_classifications')
      .select(`
        *,
        classification_units(
          unit_id,
          is_default,
          units:unit_id(code, name)
        ),
        classification_brands(
          brand_id,
          is_default,
          brands:brand_id(name_ar, name_en, logo_url)
        )
      `)
      .eq('sector_id', sectorId)
      .eq('is_active', true)
      .order('sort_order');
    
    if (error) throw error;
    return data;
  },

  // إضافة تصنيف جديد
  async addClassification(data: {
    sector_id: string;
    name: string;
    name_ar?: string;
    description?: string;
    item_type: 'product' | 'waste' | 'both';
  }) {
    const { data: result, error } = await supabase
      .from('unified_classifications')
      .insert(data)
      .select()
      .single();
    
    if (error) throw error;
    return result;
  },

  // ربط وحدة بتصنيف
  async linkUnitToClassification(classificationId: string, unitId: number, isDefault: boolean = false) {
    const { data, error } = await supabase
      .from('classification_units')
      .insert({
        classification_id: classificationId,
        unit_id: unitId,
        is_default: isDefault
      })
      .select()
      .single();
    
    if (error) throw error;
    return data;
  },

  // ربط براند بتصنيف
  async linkBrandToClassification(classificationId: string, brandId: string, isDefault: boolean = false) {
    const { data, error } = await supabase
      .from('classification_brands')
      .insert({
        classification_id: classificationId,
        brand_id: brandId,
        is_default: isDefault
      })
      .select()
      .single();
    
    if (error) throw error;
    return data;
  }
};
```
