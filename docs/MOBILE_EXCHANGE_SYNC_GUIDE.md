# دليل مزامنة البورصة لتطبيق الموبايل (محدث)

هذا المستند يوضح لمبرمجي الموبايل كيفية الحصول على أسعار المخلفات اللحظية بعد أتمتة نظام التسعير في لوحة التحكم.

## 1. مصدر البيانات الوحيد (Single Source of Truth)

**يجب التوقف عن القراءة من جداول الكتالوج (`subcategories`) لأغراض التسعير.**
المصدر الوحيد والصحيح لأسعار البورصة هو جدول:
`stock_exchange`

## 2. كيفية عرض شريط البورصة (Ticker)

لعرض شريط التمرير العلوي في تطبيق الموبايل، استخدم الاستعلام التالي:

```sql
SELECT 
    se.id,
    se.buy_price as current_price,
    wda.name as material_name,
    wsc.name as subcategory_name,
    se.show_on_ticker
FROM stock_exchange se
JOIN waste_data_admin wda ON se.product_id = wda.id
JOIN waste_sub_categories wsc ON se.sub_category_id = wsc.id
WHERE se.show_on_ticker = true
ORDER BY se.last_update DESC;
```

### حقول هامة:
- `buy_price`: السعر النهائي للكيلو الذي يظهر للمستخدم.
- `show_on_ticker`: عرض المادة فقط إذا كانت هذه القيمة `true`.

## 3. التحديث اللحظي (Real-time Updates)

لوحة التحكم تقوم الآن بتحديث جدول `stock_exchange` تلقائياً في الحالات التالية:
1. **تحديث سعر الفئة**: عند تغيير سعر "الكرتون" مثلاً، يتم تحديث كافة المنتجات التابعة له (كرتون مضلع، كرتون ميكس، إلخ) في جدول البورصة فوراً.
2. **تعديل مادة فردية**: عند تغيير "نسبة التميز" لمنتج معين، يتم تحديث سعره في البورصة فوراً.
3. **إضافة منتج جديد**: بمجرد إضافة منتج في لوحة التحكم، يتم إنشاء سجل له في البورصة تلقائياً.

**نصيحة**: يفضل الاشتراك في Real-time channel لجدول `stock_exchange` في تطبيق الموبايل ليعكس تغير الأسعار أمام المستخدم دون الحاجة لإعادة تحميل التطبيق.

## 4. حل مشكلة "السعر غير متاح"

إذا ظهر السعر غير متاح لمادة معينة، فهذا يعني غالباً أنها لم يتم "نشرها" أو لم يتم تحديد "سعر الفئة" الخاصة بها في لوحة التحكم. النظام الجديد يضمن إنشاء السجلات تلقائياً بمجرد إدخال الأسعار في الحاسبة الذكية.
