# وثيقة تطوير محرك الذكاء السلوكي (Behavioral Intelligence Engine - V2.2)

## النسخة النهائية: السيادة السلوكية والتعلم المستمر (Behavioral Sovereignty & Learning)

---

## 1. الرؤية المحورية (The Final Vision)

تحويل المنصة إلى "بيئة نمو ذكية" حيث لا يقتصر دور الناشر على وضع المحتوى، بل يمتد
إلى تصميم الأثر النفسي المستهدف. النظام يضمن الدقة عبر مراحل متدرجة من الإفصاح
(Progressive Disclosure) ويحمي الخصوصية عبر ميثاق أخلاقي صارم.

---

## 2. هندسة واجهة النشر الذكية (UI/UX Engineering)

### أ- التدرج في الإفصاح (Progressive Disclosure)

لتجنب إرهاق الناشر، تنقسم الواجهة إلى مستويين:

1. **المستوى 1 (للمبتدئين)**: اختيار "نوع المنشور" من القوالب الجاهزة (تعليمي،
   شخصي، ملهم).
2. **المستوى 2 (للمتقدمين)**: فتح "لوحة التحكم الدقيقة" لضبط النسب يدوياً (فكري،
   اجتماعي، قيمي) مع التحقق من قاعدة الـ 100%.

### ب- القوالب الذكية (Smart Templates)

- **📚 التعليمي**: (70% فكري، 10% اجتماعي، 20% قيمي).
- **💬 القصصي**: (10% فكري, 70% اجتماعي، 20% قيمي).
- **✨ الإلهامي**: (20% فكري، 20% اجتماعي، 60% قيمي).

---

## 3. محرك الجودة والتصنيف (Quality & Analytics)

### أ- حساب مستوى الثقة (Automatic Confidence Scoring)

بدلاً من الاختيار اليدوي، يحسب النظام الثقة تلقائياً:

- **مرتفع (High)**: عند استخدام قالب جاهز أو تعديل طفيف (±5%).
- **متوسط (Medium)**: عند التعديل اليدوي الواسع المتوازن.
- **منخفض (Low)**: عند اختيار "لا أعرف" (تحويل لـ Admin Review) أو توزيع غير
  تقليدي.

### ب- التلعيب والتحفيز (Gamification)

منح أوسمة رقمية للناشرين المتميزين سلوكياً:

- **🎯 الناشر الدقيق (Accurate Publisher)**: لمن يحقق تطابقاً >85% بين أهدافه
  وتفاعل الجمهور.
- **⚖️ المبدع المتوازن (Balanced Creator)**: لمن ينوع محتواه ليشمل الأبعاد
  الثلاثة.

---

## 4. حلقة التعلم (Publisher Learning Loop)

بعد مرور فترة على النشر، تظهر للناشر لوحة "تحليل الأثر الفعلي":

- **Audience Match**: "منشورك استهدف العقول المفكرة، ولكنه جذب الشخصيات العاطفية
  بنسبة تزيد عن توقعك بـ 30%".
- **Growth Opportunities**: "جمهورك حالياً متعطش للمحتوى القيمي، جرب كتابة نصيحة
  في منشورك القادم".

## 5. تطوير هيكلة البيانات (Technical Schema)

### أ- جدول المنشورات (Posts Table)

```sql
ALTER TABLE posts ADD COLUMN 
  intellectual_pct INTEGER,
  social_pct INTEGER,
  values_pct INTEGER,
  template_used VARCHAR(50),      -- القالب المستخدم
  classification_source ENUM(      -- مصدر التصنيف
    'publisher_manual',
    'publisher_template', 
    'admin_review',
    'system_neutral'
  ),
  accuracy_score DECIMAL(3,2),    -- دقة توقع الناشر
  confidence_level VARCHAR(20);   -- مستوى الثقة المحسوب
```

### ب- جدول سجل تعلم الناشر (Publisher Learning History)

```sql
CREATE TABLE publisher_learning (
  id UUID PRIMARY KEY,
  publisher_id UUID,
  post_id UUID,
  predicted_profile JSONB,   -- التوقع (فكري/اجتماعي/قيمي)
  actual_profile JSONB,      -- الواقع (سمات المتفاعلين الفعلية)
  accuracy_score DECIMAL(3,2),
  created_at TIMESTAMP
);
```

## 6. الميثاق الأخلاقي والخصوصية (Privacy & Ethics)

هذا النظام يلتزم بمبدأ **"السيادة السلوكية للعميل"**:

1. **لوحة الشفافية**: يمكن للعميل رؤية ملفه النفسي وكيفية بنائه.
2. **خيار الانسحاب (Opt-out)**: يحق للعميل إيقاف التحليل النفسي لسلوكه في أي وقت.
3. **الاستخدام النزيه**: البيانات تستخدم لترتيب المحتوى وتحسين التجربة، وليس
   للتلاعب بالقرارات المالية أو الشخصية.
