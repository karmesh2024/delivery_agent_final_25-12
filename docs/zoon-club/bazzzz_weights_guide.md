# دليل توزيع الأوزان النفسية لأنواع Bazzzz (V1.0)

هذا المستند يمثل المرجع الأساسي لإدارة محرك الذكاء السلوكي في نادي زون. يحدد هذا
الدليل كيف يؤثر كل "تفاعل" يقوم به العضو على ملفه الشخصي وفق نموذج **السمات
الخمس الكبرى (Big Five)**.

## 🧠 مرجع السمات (The Big Five)

1. **Openness (O)**: الانفتاح على الأفكار والإبداع.
2. **Conscientiousness (C)**: الانضباط، الدقة، وإتمام المهام.
3. **Extraversion (E)**: الطاقة الاجتماعية والانبساط.
4. **Agreeableness (A)**: الوفاق، التعاطف، والتعاون.
5. **Neuroticism (N)**: (القيم السالبة تعني استقرار عاطفي، الموجبة تعد
   حساسبة/قلق).

## 🛠 تعليمات الإدارة (Admin Notes)

- **ثبات النظام**: لا تقم بتعديل الأوزان بشكل متكرر لضمان دقة ملفات الأعضاء.
- **التدرج**: القيمة 0 تعني تجاهل السمة، والقيمة 10 تعني تأثيراً جوهرياً.
- **التوازن**: مجموع الأوزان لكل تفاعل مصمم ليكون بين 10-20 نقطة لضمان استقرار
  التحليل.
