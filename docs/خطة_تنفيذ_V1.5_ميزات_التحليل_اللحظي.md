## 📋 نظرة عامة

في التحديث V1.5، أصبح الداش بورد شريكاً تفاعلياً للموبايل. التطبيق لا يرسل فقط الوقت، بل يرسل حالة "المحرك" (Audio Engine State) وتفاصيل الجلسة الصافية.

### الفرق بين V1.4 و V1.5:

| الميزة | V1.4 | V1.5 |
|--------|------|------|
| **duration_minutes** | دقائق إجمالية من وقت البدء | دقائق صافية (بعد خصم أوقات التوقف) ✅ |
| **points_earned** | نقاط متوقعة | نقاط مستحقة حتى هذه اللحظة ✅ |
| **is_playing** | غير موجود | حالة التشغيل اللحظية (يعمل/متوقف) ✅ |
| **Session Tracking** | غير متاح | تتبع الجلسات الفريدة ✅ |

### 2. تحديث RPC Function `get_active_radio_listeners`

```sql
-- Migration: 20250122_update_get_active_radio_listeners_v1_5.sql
DROP FUNCTION IF EXISTS get_active_radio_listeners(UUID);

CREATE OR REPLACE FUNCTION get_active_radio_listeners(p_activity_id UUID)
RETURNS TABLE (
  id UUID,
  user_id UUID,
  activity_id UUID,
  started_listening_at TIMESTAMPTZ,
  last_active_at TIMESTAMPTZ,
  duration_minutes INTEGER,
  points_earned INTEGER,
  is_playing BOOLEAN,  -- ✅ V1.5: جديد
  is_active BOOLEAN,
  created_at TIMESTAMPTZ,
  updated_at TIMESTAMPTZ,
  location JSONB
)
LANGUAGE sql
SECURITY DEFINER
STABLE
AS $$
  SELECT 
    l.id,
    l.user_id,
    l.activity_id,
    l.started_listening_at,
    l.last_active_at,
    l.duration_minutes,  -- V1.5: دقائق صافية (بعد خصم أوقات التوقف)
    l.points_earned,     -- V1.5: نقاط مستحقة حتى هذه اللحظة
    l.is_playing,        -- ✅ V1.5: حالة التشغيل اللحظية
    l.is_active,
    l.created_at,
    l.updated_at,
    l.location
  FROM radio_listeners l
  WHERE l.activity_id = p_activity_id
    AND l.is_active = true
    AND l.last_active_at > NOW() - INTERVAL '5 minutes'
  ORDER BY l.last_active_at DESC;
$$;

COMMENT ON FUNCTION get_active_radio_listeners IS 'V1.5: جلب المستمعين النشطين - duration_minutes (صافية), points_earned (مستحقة), is_playing (حالة التشغيل)';
```

```typescript
// ✅ V1.5: دالة مساعدة لتحديد حالة المستمع
const getListenerStatus = (listener: RadioListenerWithDetails) => {
```

### 3. تحديث Real-time Subscription

**الملف:** `src/app/club-zone/radio/page.tsx`

```typescript
// ✅ V1.5: تحديث Real-time subscription لتشمل is_playing
useEffect(() => {
  if (!currentStream?.id) return;
  
  const channel = supabase
    .channel(`radio_listeners_${currentStream.id}`)
    .on(
      'postgres_changes',
      { 
        event: 'UPDATE', 
        schema: 'public', 
        table: 'radio_listeners', 
        filter: `activity_id=eq.${currentStream.id}` 
      },
      (payload) => {
        const updatedListener = payload.new as any;
        setActiveListeners((prev) => {
          const index = prev.findIndex((l) => l.id === updatedListener.id);
          if (index === -1) return prev;
          
          const updated = [...prev];
          updated[index] = {
            ...updated[index],
            duration_minutes: updatedListener.duration_minutes || 0,
            points_earned: updatedListener.points_earned || 0,
            is_playing: updatedListener.is_playing ?? true,  // ✅ V1.5: تحديث حالة التشغيل
            last_active_at: updatedListener.last_active_at,
            updated_at: updatedListener.updated_at,
          };
          return updated;
        });
      }
    )
    .subscribe((status) => console.log('[Radio Realtime] Subscription status:', status));
  
  return () => {
    console.log('[Radio Realtime] Cleaning up subscription');
    supabase.removeChannel(channel);
  };
}, [currentStream?.id]);
```

### 🟢 د - إحصائيات متقدمة (Advanced Statistics)

**الوصف:** عرض إحصائيات إضافية مثل:
- عدد المستمعين النشطين (يستمعون الآن)
- عدد المستمعين المتوقفين مؤقتاً
- متوسط وقت الاستماع
- إجمالي النقاط المستحقة

**التنفيذ:**

```typescript
const stats = useMemo(() => {
  const active = activeListeners.filter(l => l.is_active);
  const playing = active.filter(l => l.is_playing);
  const paused = active.filter(l => !l.is_playing);
  const totalPoints = active.reduce((sum, l) => sum + (l.points_earned || 0), 0);
  const avgDuration = active.length > 0
    ? Math.round(active.reduce((sum, l) => sum + (l.duration_minutes || 0), 0) / active.length)
    : 0;

  return {
    total: active.length,
    playing: playing.length,
    paused: paused.length,
    totalPoints,
    avgDuration,
  };
}, [activeListeners]);

// في JSX:
<Card>

      <div>
        <p className="text-sm text-muted-foreground">إجمالي المستمعين</p>
        <p className="text-2xl font-bold">{stats.total}</p>
      </div>
      <div>
```
