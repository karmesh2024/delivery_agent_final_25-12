## 📝 خطوات الشراء (دليل كامل)

### الخطوة 1: إنشاء حساب

1. **الذهاب إلى الموقع:**
   - الرابط: https://console.hetzner.cloud/
   - أو: https://www.hetzner.com/cloud

2. **إنشاء حساب جديد:**
   - اضغط "Sign Up" أو "Create Account"
   - أدخل:
     - البريد الإلكتروني
     - كلمة المرور
     - الاسم
     - العنوان (للفوترة)

3. **التحقق من البريد:**
   - ستصلك رسالة تأكيد
   - اضغط على الرابط للتأكيد

4. **التحقق من الهوية (اختياري لكن موصى به):**
   - قد يطلبون التحقق من الهوية (لحماية من الاحتيال)
   - يمكنك إرسال صورة من بطاقة الهوية أو جواز السفر

---

### الخطوة 2: إضافة رصيد (Deposit)

1. **الدخول إلى Console:**
   - https://console.hetzner.cloud/

2. **إضافة رصيد:**
   - اضغط على "Billing" في القائمة
   - اضغط "Add Credit"
   - أدخل المبلغ (مثلاً €10-20 للبداية)
   - طرق الدفع:
     - Credit Card (Visa/Mastercard)
     - PayPal
     - Bank Transfer (للمبالغ الكبيرة)

**ملاحظة:** يمكنك البدء بدون إضافة رصيد (Pay-as-you-go)

---

### الخطوة 3: إنشاء VPS

1. **الدخول إلى Cloud Console:**
   - https://console.hetzner.cloud/

2. **إنشاء Server جديد:**
   - اضغط "Add Server" أو "New Server"
   - اختر:
     - **Location**: Nuremberg (nbg1) أو Falkenstein (fsn1)
     - **Image**: Ubuntu 22.04 (أو 24.04)
     - **Type**: CX11 (للبداية) أو CX22 (أفضل)
     - **SSH Key**: (اختياري - للأمان)
     - **Name**: "karmesh-radio" أو "radio-server"

3. **المراجعة والإنشاء:**
   - راجع المواصفات
   - اضغط "Create & Buy Now"

4. **الحصول على معلومات الاتصال:**
   - بعد الإنشاء (30-60 ثانية)
   - ستحصل على:
     - **IP Address**: (مثلاً 123.45.67.89)
     - **Root Password**: (احفظها في مكان آمن!)

---

### الخطوة 4: الاتصال بـ VPS

**من Windows (PowerShell أو CMD):**

```bash
# تثبيت SSH (إذا لم يكن مثبتاً)
# Windows 10/11 لديه SSH مدمج

# الاتصال
ssh root@YOUR_IP_ADDRESS

# سيطلب كلمة المرور (Root Password)
```

**من Mac/Linux:**

```bash
ssh root@YOUR_IP_ADDRESS
```

## 🔒 الأمان والإعدادات المهمة

### 1. إعداد Firewall

```bash
# بعد الاتصال بـ VPS
sudo ufw allow 22/tcp    # SSH
sudo ufw allow 80/tcp    # HTTP
sudo ufw allow 443/tcp   # HTTPS
sudo ufw enable
```

### 2. إعداد SSH Key (بدلاً من Password)

```bash
# على جهازك المحلي
ssh-keygen -t rsa -b 4096

# نسخ المفتاح إلى VPS
ssh-copy-id root@YOUR_IP_ADDRESS
```

### 3. تعطيل Root Login (اختياري)

```bash
# إنشاء مستخدم جديد
adduser karmesh
usermod -aG sudo karmesh

# تعطيل Root Login
sudo nano /etc/ssh/sshd_config
# PasswordAuthentication no
# PermitRootLogin no
```
