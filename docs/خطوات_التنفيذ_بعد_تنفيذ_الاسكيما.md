## 👥 الخطوة 3: ربط الصلاحيات بالأدوار

### 3.1 التحقق من ربط الصلاحيات
```sql
-- التحقق من ربط الصلاحيات بالأدوار
SELECT 
  r.name AS role_name,
  p.code AS permission_code,
  p.name AS permission_name
FROM public.role_permissions rp
JOIN public.roles r ON rp.role_id = r.id
JOIN public.permissions p ON rp.permission_id = p.id
WHERE p.code LIKE 'waste:%'
ORDER BY r.name, p.code;
```

### 3.2 ربط يدوي (إذا لم يتم الربط تلقائياً)
```sql
-- مثال: ربط صلاحيات بدور معين
-- استبدل 'YOUR_ROLE_ID' بمعرف الدور المطلوب

-- جلب معرف الدور
SELECT id, name FROM public.roles WHERE name = 'admin';

-- ربط الصلاحيات
INSERT INTO public.role_permissions (role_id, permission_id, created_at)
SELECT 
  'YOUR_ROLE_ID'::uuid,  -- استبدل بمعرف الدور
  id,
  NOW()
FROM public.permissions
WHERE code LIKE 'waste:%'
ON CONFLICT (role_id, permission_id) DO NOTHING;
```

### 5.1 إنشاء دور "مدير المخلفات"
```sql
-- إنشاء دور جديد لمديري المخلفات
INSERT INTO public.roles (name, code, description, is_active)
VALUES (
  'Waste Manager',
  'waste_manager',
  'مدير إدارة المخلفات - يتحكم في جميع عمليات المخلفات',
  true
) ON CONFLICT (code) DO NOTHING
RETURNING id;

-- ربط جميع صلاحيات المخلفات بهذا الدور
INSERT INTO public.role_permissions (role_id, permission_id, created_at)
SELECT 
  (SELECT id FROM public.roles WHERE code = 'waste_manager'),
  id,
  NOW()
FROM public.permissions
WHERE code LIKE 'waste:%'
ON CONFLICT (role_id, permission_id) DO NOTHING;
```
