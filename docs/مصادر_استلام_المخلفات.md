# مصادر استلام المخلفات

## 📋 نظرة عامة

صفحة **طلبات الاستلام** في إدارة المخلفات تشمل **جميع مصادر الاستلام** وليس فقط الدليفري بوي. النظام يدعم مصادر متعددة للاستلام.

---

## 🔄 مصادر الاستلام

### 1. **الدليفري بوي (Delivery Boys)** 🚚
- **المصدر**: `waste_collection_sessions`
- **الوصف**: الدليفري بوي يجمع المخلفات من العملاء ويسلمها للمخازن
- **الحقول**:
  - `source = 'delivery_boy'`
  - `delivery_agent_id` - معرف الدليفري بوي
  - `collection_session_id` - معرف جلسة التجميع

### 2. **الموردين (Suppliers)** 📦
- **المصدر**: `warehouse_invoices`
- **الوصف**: الموردين يوردون مخلفات مباشرة للمخازن
- **الحقول**:
  - `source = 'supplier'`
  - `supplier_id` - معرف المورد
  - `supplier_invoice_id` - معرف فاتورة المورد

### 3. **الوكلاء (Agents)** 👤
- **المصدر**: `agent_shipping_deliveries`
- **الوصف**: الوكلاء يوردون مخلفات للمخازن
- **الحقول**:
  - `source = 'agent'`
  - `agent_id` - معرف الوكيل

### 4. **مباشر (Direct)** 📥
- **المصدر**: إدخال يدوي
- **الوصف**: استلام مباشر من أي مصدر آخر (مثل استلام من جهات خارجية)
- **الحقول**:
  - `source = 'direct'`
  - لا يوجد معرفات محددة

---

## 📊 جدول `waste_receiving_approval_requests`

### الحقول الرئيسية:

```sql
CREATE TABLE waste_receiving_approval_requests (
  id UUID PRIMARY KEY,
  source TEXT NOT NULL, -- 'delivery_boy', 'supplier', 'agent', 'direct'
  
  -- حقول مشروطة حسب المصدر
  collection_session_id UUID, -- إذا كان source = 'delivery_boy'
  supplier_invoice_id INT,    -- إذا كان source = 'supplier'
  delivery_agent_id UUID,      -- إذا كان source = 'delivery_boy'
  supplier_id INT,            -- إذا كان source = 'supplier'
  agent_id UUID,              -- إذا كان source = 'agent'
  
  warehouse_id INT NOT NULL,  -- المخزن المستلم (مشترك لجميع المصادر)
  waste_items JSONB NOT NULL, -- المخلفات المستلمة
  status TEXT,                 -- pending_verification, verified, approved, rejected
  ...
);
```

---

## 🔄 التدفق حسب المصدر

### 1. استلام من الدليفري بوي:
```
الدليفري بوي يجمع من العملاء
    ↓
إنشاء waste_collection_sessions
    ↓
الدليفري بوي يسلم للمخزن
    ↓
إنشاء طلب استلام (source = 'delivery_boy')
    ↓
إدارة المخلفات تتحقق وتوافق
```

### 2. استلام من الموردين:
```
المورد يورد مخلفات للمخزن
    ↓
إنشاء warehouse_invoices
    ↓
إنشاء طلب استلام (source = 'supplier')
    ↓
إدارة المخلفات تتحقق وتوافق
```

### 3. استلام من الوكلاء:
```
الوكيل يورد مخلفات للمخزن
    ↓
إنشاء agent_shipping_deliveries
    ↓
إنشاء طلب استلام (source = 'agent')
    ↓
إدارة المخلفات تتحقق وتوافق
```

### 4. استلام مباشر:
```
إدخال يدوي من إدارة المخلفات
    ↓
إنشاء طلب استلام (source = 'direct')
    ↓
إدارة المخلفات تتحقق وتوافق
```

## 💡 الفوائد

### 1. **مركزية شاملة**
- ✅ نقطة واحدة لجميع مصادر الاستلام
- ✅ تتبع موحد لجميع المخلفات الواردة

### 2. **مرونة عالية**
- ✅ دعم مصادر متعددة
- ✅ سهولة إضافة مصادر جديدة

### 3. **تتبع كامل**
- ✅ تتبع مصدر كل شحنة
- ✅ ربط مع الجداول الأصلية (sessions, invoices, etc.)

## 🎯 الاستخدام

### إنشاء طلب استلام من الدليفري بوي:
```typescript
await receivingApprovalService.createReceivingRequest({
  source: 'delivery_boy',
  delivery_agent_id: agentId,
  collection_session_id: sessionId,
  warehouse_id: warehouseId,
  waste_items: [...],
});
```

### إنشاء طلب استلام من المورد:
```typescript
await receivingApprovalService.createReceivingRequest({
  source: 'supplier',
  supplier_id: supplierId,
  supplier_invoice_id: invoiceId,
  warehouse_id: warehouseId,
  waste_items: [...],
});
```

### إنشاء طلب استلام من الوكيل:
```typescript
await receivingApprovalService.createReceivingRequest({
  source: 'agent',
  agent_id: agentId,
  warehouse_id: warehouseId,
  waste_items: [...],
});
```

### إنشاء طلب استلام مباشر:
```typescript
await receivingApprovalService.createReceivingRequest({
  source: 'direct',
  warehouse_id: warehouseId,
  waste_items: [...],
});
```

## 📝 ملاحظات مهمة

1. **المصدر إلزامي**: يجب تحديد `source` لكل طلب استلام
2. **الحقول المشروطة**: الحقول الأخرى تعتمد على نوع المصدر
3. **المخزن مشترك**: `warehouse_id` مطلوب لجميع المصادر
4. **التحقق والموافقة**: نفس العملية لجميع المصادر

---

## ✅ الخلاصة

صفحة طلبات الاستلام تشمل:
- ✅ **الدليفري بوي**: يجمع من العملاء
- ✅ **الموردين**: يوردون مباشرة للمخازن
- ✅ **الوكلاء**: يوردون للمخازن
- ✅ **مباشر**: أي مصدر آخر

جميع المصادر تمر بنفس العملية: **التحقق → الموافقة → تحديث المخزون**
