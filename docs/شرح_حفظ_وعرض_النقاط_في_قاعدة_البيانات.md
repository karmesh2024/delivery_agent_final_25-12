# شرح حفظ وعرض النقاط في قاعدة البيانات

## السؤال: أين يتم حفظ النقاط عند إيقاف التشغيل وكيف يتم عرضها في صفحة إدارة نقاط النادي؟

---

## 📊 الجداول المستخدمة

### 1. جدول `club_points_wallet` (محفظة النقاط)

**الوصف:** يحتوي على رصيد النقاط الإجمالي للمستخدم

```sql
CREATE TABLE club_points_wallet (
  id UUID PRIMARY KEY,
  user_id UUID NOT NULL UNIQUE REFERENCES new_profiles(id),
  points_balance INTEGER DEFAULT 0,        -- الرصيد الإجمالي (للتوافق القديم)
  lifetime_points INTEGER DEFAULT 0,       -- إجمالي النقاط المكتسبة مدى الحياة
  pending_points INTEGER DEFAULT 0,        -- نقاط قيد الاعتماد (رصيد هذا الشهر)
  available_points INTEGER DEFAULT 0,      -- نقاط جاهزة للاستخدام (رصيد متاح)
  used_points INTEGER DEFAULT 0,           -- نقاط مستخدمة
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

**الحقول المهمة:**
- `pending_points`: النقاط المكتسبة من الراديو تذهب هنا أولاً ✅
- `available_points`: النقاط المعتمدة والجاهزة للاستخدام
- `used_points`: النقاط المستخدمة في الجوائز
- `lifetime_points`: إجمالي النقاط المكتسبة مدى الحياة

### 2. جدول `club_points_transactions` (معاملات النقاط) ✅ **هذا هو الجدول المعروض في الصفحة**

**الوصف:** يحتوي على سجل جميع معاملات النقاط (كسب، استخدام، اعتماد)

```sql
CREATE TABLE club_points_transactions (
  id UUID PRIMARY KEY,
  user_id UUID NOT NULL REFERENCES new_profiles(id),
  transaction_type VARCHAR(20) NOT NULL CHECK (
    transaction_type IN ('EARNED', 'ACTIVATED', 'USED')
  ),
  points INTEGER NOT NULL,                  -- القيمة (موجب للنقاط المكتسبة)
  points_before INTEGER,                    -- الرصيد قبل المعاملة
  points_after INTEGER,                     -- الرصيد بعد المعاملة
  reason VARCHAR(255),                      -- السبب
  source VARCHAR(50),                       -- المصدر: 'radio_stream'
  description TEXT,                         -- وصف تفصيلي
  created_by UUID REFERENCES new_profiles(id),
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```

**الحقول المهمة:**
- `transaction_type`: نوع المعاملة (`EARNED`, `ACTIVATED`, `USED`)
- `points`: عدد النقاط (موجب للكسب، سالب للاستخدام)
- `source`: مصدر النقاط (`'radio_stream'` للراديو)
- `description`: وصف تفصيلي (مثال: "نقاط من الاستماع للراديو: 30 دقيقة")

### الخطوة 2: RPC Function `stop_user_listening_session_and_award`

```sql
CREATE OR REPLACE FUNCTION stop_user_listening_session_and_award(
  p_session_id UUID,
  p_user_id UUID,
  p_activity_id UUID
)
RETURNS JSONB
AS $$
DECLARE
  v_duration_minutes INTEGER;
  v_points_earned INTEGER;
BEGIN
  -- 1) حساب المدة
  v_duration_minutes := FLOOR(EXTRACT(EPOCH FROM (NOW() - start_time)) / 60);
  
  -- 2) حساب النقاط
  v_points_earned := calculate_service_points('radio_stream', v_duration_minutes);
  
  -- 3) تحديث radio_user_sessions
  UPDATE radio_user_sessions
  SET 
    stop_time = NOW(),
    duration_minutes = v_duration_minutes,
    points_earned = v_points_earned,
    status = 'closed'
  WHERE id = p_session_id;
  
  -- 4) ✅ منح النقاط (هنا يتم الحفظ!)
  PERFORM update_club_points_wallet(
    p_user_id,
    v_points_earned,
    'EARNED',                    -- نوع المعاملة
    'استماع راديو كارمش',        -- السبب
    'radio_stream',              -- المصدر
    'نقاط من الاستماع للراديو: ' || v_duration_minutes || ' دقيقة',  -- الوصف
    NULL                          -- created_by
  );
  
  RETURN jsonb_build_object(
    'minutes_listened', v_duration_minutes,
    'points_earned', v_points_earned
  );
END;
$$;
```

### الخطوة 3: RPC Function `update_club_points_wallet` ✅ **هنا يتم الحفظ الفعلي**

```sql
CREATE OR REPLACE FUNCTION update_club_points_wallet(
  p_user_id UUID,
  p_points INTEGER,
  p_transaction_type VARCHAR,  -- 'EARNED'
  p_reason VARCHAR,            -- 'استماع راديو كارمش'
  p_source VARCHAR,            -- 'radio_stream'
  p_description TEXT,          -- 'نقاط من الاستماع للراديو: 30 دقيقة'
  p_created_by UUID
) RETURNS UUID
AS $$
DECLARE
  v_balance_before INTEGER;
  v_balance_after INTEGER;
  v_transaction_id UUID;
BEGIN
  -- 1) جلب الرصيد الحالي
  SELECT COALESCE(points_balance, 0) INTO v_balance_before
  FROM club_points_wallet
  WHERE user_id = p_user_id;
  
  -- 2) تحديث club_points_wallet
  IF p_transaction_type = 'EARNED' THEN
    UPDATE club_points_wallet
    SET 
      pending_points = pending_points + p_points,      -- ✅ إضافة إلى pending_points
      available_points = available_points + p_points,  -- ✅ إضافة إلى available_points
      points_balance = available_points + p_points,     -- ✅ تحديث الرصيد الإجمالي
      lifetime_points = lifetime_points + p_points,     -- ✅ تحديث إجمالي مدى الحياة
      updated_at = NOW()
    WHERE user_id = p_user_id;
    
    v_balance_after := v_balance_before + p_points;
  END IF;
  
  -- 3) ✅ إدراج سجل في club_points_transactions (هذا ما يظهر في الجدول!)
  INSERT INTO club_points_transactions (
    user_id,
    transaction_type,        -- 'EARNED'
    points,                   -- 30 (مثال)
    points_before,            -- الرصيد قبل المعاملة
    points_after,             -- الرصيد بعد المعاملة
    reason,                   -- 'استماع راديو كارمش'
    source,                   -- 'radio_stream'
    description,              -- 'نقاط من الاستماع للراديو: 30 دقيقة'
    created_by
  ) VALUES (
    p_user_id,
    p_transaction_type,
    p_points,
    v_balance_before,
    v_balance_after,
    p_reason,
    p_source,
    p_description,
    p_created_by
  ) RETURNING id INTO v_transaction_id;
  
  RETURN v_transaction_id;
END;
$$;
```

## 📋 ملخص: أين يتم الحفظ؟

### ✅ عند إيقاف التشغيل، يتم حفظ البيانات في:

1. **`club_points_wallet`** ✅
   - `pending_points`: يتم زيادة النقاط المكتسبة
   - `available_points`: يتم زيادة النقاط المكتسبة
   - `points_balance`: يتم تحديث الرصيد الإجمالي
   - `lifetime_points`: يتم تحديث إجمالي مدى الحياة

2. **`club_points_transactions`** ✅ **هذا هو الجدول المعروض في الصفحة**
   - يتم إدراج سجل جديد لكل معاملة
   - يحتوي على: `transaction_type = 'EARNED'`, `source = 'radio_stream'`, `description = 'نقاط من الاستماع للراديو: X دقيقة'`

3. **`radio_user_sessions`** (مرجع تاريخي)
   - `duration_minutes`: المدة الفعلية
   - `points_earned`: النقاط المكتسبة
   - `status = 'closed'`
