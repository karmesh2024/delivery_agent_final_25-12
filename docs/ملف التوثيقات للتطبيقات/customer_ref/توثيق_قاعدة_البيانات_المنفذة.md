## 5. العلاقات الرئيسية بين الجداول

### 5.1 علاقة العملاء بالملفات الشخصية

- **العلاقة**: جدول `customers` مرتبط بجدول `new_profiles` عبر علاقة واحد لواحد.
- **المفتاح**: يتم استخدام نفس معرف المستخدم (id) في كلا الجدولين.
- **الغرض**: فصل بيانات المصادقة عن معلومات الملف الشخصي.

### 5.2 علاقة الملفات الشخصية بالعناوين

- **العلاقة**: جدول `new_profiles` مرتبط بجدول `customer_addresses` عبر علاقة واحد لكثير.
- **المفتاح**: profile_id في جدول `customer_addresses` يرتبط بـ id في جدول `new_profiles`.
- **الغرض**: تمكين العميل من امتلاك عدة عناوين.

### 5.3 علاقة الملفات الشخصية بأرقام الهواتف

- **العلاقة**: جدول `new_profiles` مرتبط بجدول `customer_phones` عبر علاقة واحد لكثير.

## 7. ملخص الاختلافات بين قاعدة البيانات والتطبيق الحالي

### 7.1 اختلافات الإنومز

1. **CustomerStatus**:
   - قاعدة البيانات: `active`, `inactive`, `suspended`, `blocked`
   - التطبيق: `active`, `inactive`, `suspended`
   - **النقص**: قيمة `blocked` غير موجودة في التطبيق

2. **verification_status_enum / phone_verification_status**:
   - قاعدة البيانات: `pending`, `verified`, `failed`, `expired`, `max_attempts`
   - التطبيق: `pending`, `verified`, `failed`
   - **النقص**: قيمتي `expired` و `max_attempts` غير موجودتين في التطبيق

3. **CustomerOrderStatusEnum / delivery_status_enum**:
   - قاعدة البيانات: `pending`, `confirmed`, `inProgress`, `completed`, `cancelled/canceled`
   - التطبيق: `pending`, `confirmed`, `pickedUp`, `inReceipt`, `completed`, `cancelled`, `scheduled`, `returned`
   - **الاختلاف**: اختلاف في المسميات وإضافة قيم في التطبيق

4. **PaymentMethodEnum**:
   - قاعدة البيانات: `cash`, `credit`, `wallet`
   - التطبيق: `cash`, `wallet`, `credit_card`, `online_payment`
   - **الاختلاف**: اختلاف في المسميات وإضافة قيم في التطبيق

### 7.2 نواحي التحسين المقترحة

1. **توحيد الإنومز**:
   - إضافة قيمة `blocked` إلى `CustomerStatus`
   - إضافة قيمتي `expired` و `max_attempts` إلى `verification_status_enum`
   - توحيد مسميات حالات الطلب بين التطبيق وقاعدة البيانات

2. **تحسين التوثيق**:
   - إضافة تعليقات توضيحية لكل إنوم

   - إضافة قيود المفاتيح الخارجية المفقودة
   - توحيد أسماء الحقول المتشابهة

4. **تحسين الأداء**:
