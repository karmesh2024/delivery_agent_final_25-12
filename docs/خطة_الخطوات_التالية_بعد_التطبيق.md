### 1.1 التحقق من الجداول

```sql
-- التحقق من الربحية
SELECT column_name 
FROM information_schema.columns 
WHERE table_name = 'waste_collection_items' 
  AND column_name IN ('sell_price', 'item_profit');

SELECT column_name 
FROM information_schema.columns 
WHERE table_name = 'waste_collection_sessions' 
  AND column_name IN ('buy_total', 'sell_total', 'platform_profit', 'profit_margin');

-- التحقق من الاحتياطيات
SELECT * FROM financial_reserves LIMIT 1;

-- التحقق من حدود السحب

-- التحقق من قواعد store_points

```

### 1.2 اختبار Functions

```sql
-- اختبار حساب الاحتياطيات

-- عرض النتائج
SELECT * FROM financial_reserves

-- اختبار حدود السحب

    'customer_id_here'::UUID,
    100.00
);

-- اختبار قواعد store_points
SELECT public.calculate_store_points_bonus(
    'customer_id_here'::UUID,
    10000
);
```

### 1.3 التحقق من Triggers

```sql
-- التحقق من Triggers
SELECT 
    trigger_name,
    event_manipulation,
    event_object_table
FROM information_schema.triggers
WHERE trigger_schema = 'public'
  AND trigger_name IN (
    'trigger_calculate_collection_item',
    'trigger_update_session_totals',
```

## ⚙️ المرحلة 6: إعداد Cron Job (يوم 8)

### 6.1 Cron Job للاحتياطيات اليومية

**إذا كان لديك pg_cron:**

```sql
SELECT cron.schedule(
    'daily_reserves_calculation',
    '0 0 * * *',  -- كل يوم في منتصف الليل
    'SELECT public.calculate_daily_reserves()'
);
```

**أو عبر Vercel Cron:**

**الملف:** `vercel.json`

```json
{
  "crons": [
    {
      "path": "/api/cron/daily-reserves",
      "schedule": "0 0 * * *"
    }
  ]
}
```

**الملف:** `src/app/api/cron/daily-reserves/route.ts`

```typescript
export async function GET(request: Request) {
  // التحقق من Secret
  if (request.headers.get('authorization') !== `Bearer ${process.env.CRON_SECRET}`) {
    return new Response('Unauthorized', { status: 401 });
  }
  
  // تشغيل Function
  const { data, error } = await supabase.rpc('calculate_daily_reserves');
  
  if (error) {
    return Response.json({ error: error.message }, { status: 500 });
  }
  
  return Response.json({ success: true });
}
```

```typescript
// src/app/api/admin/reserves/route.ts
import { createClient } from '@/lib/supabase/server';

export async function GET() {
  const supabase = createClient();
  
  const { data, error } = await supabase
    .from('financial_reserves')
    .select('*')
    .order('date', { ascending: false })
    .limit(1)
    .single();
  
  if (error) {
    return Response.json({ error: error.message }, { status: 500 });
  }
  
  return Response.json(data);
}
```
