# دليل نقل البورصة والشركاء الصناعيين إلى إدارة المخلفات

## 📋 نظرة عامة

هذا الدليل يوضح كيفية نقل ملفات البورصة والشركاء الصناعيين من مواقعها الحالية إلى إدارة المخلفات الجديدة.

---

## 🎯 الهدف

- نقل البورصة من `financial-management/exchange` → `waste-management/exchange`
- نقل الشركاء الصناعيين من `industrial-partners` → `waste-management/partners`
- تحديث جميع الـ imports والروابط

## 🔄 الخطوة 2: نقل ملفات البورصة

### 2.1 نقل الملفات:

**من → إلى:**
- `src/app/financial-management/exchange/page.tsx` → `src/app/waste-management/exchange/page.tsx`
- `src/app/financial-management/exchange/fallback-exchange.tsx` → `src/app/waste-management/exchange/fallback-exchange.tsx`
- `src/domains/financial-management/pages/ExchangeDashboardPage.tsx` → `src/domains/waste-management/pages/ExchangeDashboardPage.tsx`
- `src/domains/financial-management/services/exchangeService.ts` → `src/domains/waste-management/services/exchangeService.ts`
- `src/domains/financial-management/store/exchangeSlice.ts` → `src/domains/waste-management/store/exchangeSlice.ts`

### 2.2 تحديث الـ imports في الملفات المنقولة:

#### في `src/app/waste-management/exchange/page.tsx`:
```typescript
// تغيير من:

// إلى:

```

#### في `src/domains/waste-management/pages/ExchangeDashboardPage.tsx`:
```typescript
// تغيير من:
import { fetchPrices, updateExchangeProduct } from '../store/exchangeSlice';
import { StockExchange, exchangeService } from '../services/exchangeService';
import { fetchOrders } from '@/domains/industrial-partners/store/industrialPartnersSlice';

// إلى:

import { StockExchange, exchangeService } from '@/domains/waste-management/services/exchangeService';
import { fetchOrders } from '@/domains/waste-management/partners/store/industrialPartnersSlice';
import { PartnerOrder } from '@/domains/waste-management/partners/types';
```

#### في `src/domains/waste-management/services/exchangeService.ts`:
```typescript

```

#### في `src/domains/waste-management/store/exchangeSlice.ts`:
```typescript

// إلى:
```
