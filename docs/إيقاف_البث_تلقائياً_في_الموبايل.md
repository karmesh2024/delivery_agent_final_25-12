# إيقاف البث تلقائياً في تطبيق الموبايل

**التاريخ:** 2025-01-22  
**الإصدار:** V1.5.1  
**الموضوع:** إيقاف البث تلقائياً في الموبايل عند إيقافه من الداش بورد

---

## 🎯 الهدف

عند إيقاف البث من الداش بورد، يجب أن يتوقف البث تلقائياً في جميع تطبيقات الموبايل التي تستمع للراديو.

---

## 🔄 كيف يعمل النظام

### في الداش بورد:

عند الضغط على "إيقاف البث":
1. يتم تحديث `is_live = false` في جدول `club_activities`
2. يتم إرسال Real-time update إلى جميع المشتركين
3. الموبايل يتلقى التحديث ويتوقف تلقائياً

### في الموبايل:

1. **Real-time Subscription:** متابعة تحديثات `club_activities` للتحقق من `is_live`
2. **التحقق التلقائي:** عند تغيير `is_live` إلى `false`، إيقاف البث تلقائياً
3. **إعلام المستخدم:** إبلاغ المستخدم بأن البث تم إيقافه

## ✅ التنفيذ المطلوب

### 1. Real-time Subscription لحالة البث

**هذا هو الحل الأساسي:**

```dart
import 'package:supabase_flutter/supabase_flutter.dart';

class RadioStreamService {
  final SupabaseClient supabase;
  RealtimeChannel? _streamStatusChannel;
  String? _currentStreamId;
  Function(bool isLive)? _onStreamStatusChanged;
  
  RadioStreamService(this.supabase);
  
  // ✅ إعداد Real-time subscription لمتابعة حالة البث (is_live)
  void setupStreamStatusSubscription(
    String streamId, 
    Function(bool isLive) onStatusChanged
  ) {
    // إلغاء الاشتراك السابق إن وجد
    if (_streamStatusChannel != null) {
      supabase.removeChannel(_streamStatusChannel!);
    }
    
    _currentStreamId = streamId;
    _onStreamStatusChanged = onStatusChanged;
    
    _streamStatusChannel = supabase
      .channel('radio_stream_status_${streamId}')
      .onPostgresChanges(
        event: PostgresChangeEvent.update,
        schema: 'public',
        table: 'club_activities',
        filter: PostgresChangeFilter(
          type: PostgresChangeFilterType.eq,
          column: 'id',
          value: streamId,
        ),
        callback: (payload) {
          final updatedStream = payload.newRecord;
          final isLive = updatedStream['is_live'] ?? false;
          
          print('[Radio Stream] Stream status updated: is_live = $isLive');
          
          // استدعاء callback لإعلام Audio Player
          onStatusChanged(isLive);
        },
      )
      .subscribe();
    
    print('[Radio Stream] Status subscription setup for stream: $streamId');
  }
  
  // إلغاء الاشتراك
  void disposeStatusSubscription() {
    if (_streamStatusChannel != null) {
      supabase.removeChannel(_streamStatusChannel!);
      _streamStatusChannel = null;
    }
    _onStreamStatusChanged = null;
  }
}
```

### 2. معالجة إيقاف البث في Audio Player

```dart
class RadioPlayerController {
  final AudioPlayer audioPlayer;
  final RadioStreamService streamService;
  String? currentStreamId;
  bool isPlaying = false;
  bool _isStreamLive = true;
  
  RadioPlayerController(this.audioPlayer, this.streamService);
  
  // ✅ بدء الاستماع مع Real-time subscription لحالة البث
  Future<void> startListening(String streamId) async {
    currentStreamId = streamId;
    
    // 1. التحقق من حالة البث قبل البدء
    final streamStatus = await _checkStreamStatus(streamId);
    if (!streamStatus.isLive) {
      throw Exception('البث غير نشط حالياً');
    }
    
    _isStreamLive = true;
    
    // 2. قراءة الرابط الحالي من قاعدة البيانات
    final streamUrl = await getCurrentStreamUrl(streamId);
    if (streamUrl == null) {
      throw Exception('لا يوجد بث نشط');
    }
    
    // 3. بدء الاستماع
    await audioPlayer.setUrl(streamUrl);
    await audioPlayer.play();
    isPlaying = true;
    
    // 4. إعداد Real-time subscription لتحديثات الرابط
    streamService.setupStreamUrlSubscription(streamId, (newUrl) {
      _handleStreamUrlUpdate(newUrl);
    });
    
    // ✅ 5. إعداد Real-time subscription لحالة البث (is_live)
    streamService.setupStreamStatusSubscription(streamId, (isLive) {
      _handleStreamStatusUpdate(isLive);
    });
  }
  
  // ✅ معالجة تحديث حالة البث (is_live)
  Future<void> _handleStreamStatusUpdate(bool isLive) async {
    if (_isStreamLive == isLive) {
      return; // نفس الحالة، لا حاجة للتحديث
    }
    
    _isStreamLive = isLive;
    
    if (!isLive) {
      // ✅ البث تم إيقافه - إيقاف الاستماع تلقائياً
      print('[Radio Player] Stream stopped by admin - stopping playback');
      
      await audioPlayer.stop();
      isPlaying = false;
      
      // إبلاغ المستخدم
      _notifyStreamStopped();
      
      // إيقاف جلسة الاستماع في قاعدة البيانات
      await _stopListeningSession();
    } else {
      // البث تم تفعيله مرة أخرى
      print('[Radio Player] Stream resumed - can start listening again');
      // يمكن إضافة منطق لإعادة التشغيل التلقائي إذا رغبت
    }
  }
  
  // التحقق من حالة البث
  Future<StreamStatus> _checkStreamStatus(String streamId) async {
    final response = await streamService.supabase
      .from('club_activities')
      .select('is_live, is_active')
      .eq('id', streamId)
      .maybeSingle();
    
    if (response == null) {
      return StreamStatus(isLive: false, isActive: false);
    }
    
    return StreamStatus(
      isLive: response['is_live'] ?? false,
      isActive: response['is_active'] ?? true,
    );
  }
  
  // إبلاغ المستخدم بإيقاف البث
  void _notifyStreamStopped() {
    // يمكن استخدام SnackBar أو Dialog أو أي طريقة إشعار
    // مثال:
    // ScaffoldMessenger.of(context).showSnackBar(
    //   SnackBar(
    //     content: Text('تم إيقاف البث من قبل الإدارة'),
    //     duration: Duration(seconds: 5),
    //   ),
    // );
  }
  
  // إيقاف جلسة الاستماع في قاعدة البيانات
  Future<void> _stopListeningSession() async {
    // TODO: إيقاف جلسة الاستماع في radio_listeners
    // TODO: إيقاف جلسة في radio_user_sessions
  }
  
  // إيقاف الاستماع
  Future<void> stopListening() async {
    await audioPlayer.stop();
    isPlaying = false;
    _isStreamLive = false;
    streamService.dispose();
    streamService.disposeStatusSubscription();
    currentStreamId = null;
  }
  
  // قراءة الرابط الحالي من قاعدة البيانات
  Future<String?> getCurrentStreamUrl(String streamId) async {
    final response = await streamService.supabase
      .from('club_activities')
      .select('listen_url, stream_url, is_live')
      .eq('id', streamId)
      .eq('is_active', true)
      .maybeSingle();
    
    if (response == null || !(response['is_live'] ?? false)) {
      return null;
    }
    
    return response['listen_url'] ?? response['stream_url'];
  }
}

// Helper class لحالة البث
class StreamStatus {
  final bool isLive;
  final bool isActive;
  
  StreamStatus({required this.isLive, required this.isActive});
}
```

### 3. مثال كامل للاستخدام

```dart
class RadioPage extends StatefulWidget {
  @override
  _RadioPageState createState() => _RadioPageState();
}

class _RadioPageState extends State<RadioPage> {
  final AudioPlayer _audioPlayer = AudioPlayer();
  late final RadioStreamService _streamService;
  late final RadioPlayerController _playerController;
  String? _currentStreamId;
  bool _isStreamLive = false;
  
  @override
  void initState() {
    super.initState();
    _streamService = RadioStreamService(Supabase.instance.client);
    _playerController = RadioPlayerController(_audioPlayer, _streamService);
    _loadCurrentStream();
  }
  
  Future<void> _loadCurrentStream() async {
    // جلب البث الحالي
    final stream = await _getCurrentStream();
    if (stream != null) {
      setState(() {
        _currentStreamId = stream['id'];
        _isStreamLive = stream['is_live'] ?? false;
      });
    }
  }
  
  Future<Map<String, dynamic>?> _getCurrentStream() async {
    final response = await Supabase.instance.client
      .from('club_activities')
      .select()
      .eq('activity_type', 'radio_stream')
      .eq('is_active', true)
      .eq('is_live', true)
      .maybeSingle();
    
    return response;
  }
  
  Future<void> _startListening() async {
    if (_currentStreamId == null) return;
    
    if (!_isStreamLive) {
      ScaffoldMessenger.of(context).showSnackBar(
        SnackBar(
          content: Text('البث غير نشط حالياً'),
          backgroundColor: Colors.orange,
        ),
      );
      return;
    }
    
    try {
      await _playerController.startListening(_currentStreamId!);
      setState(() {});
    } catch (e) {
      print('Error starting listening: $e');
      ScaffoldMessenger.of(context).showSnackBar(
        SnackBar(
          content: Text('حدث خطأ أثناء بدء الاستماع: $e'),
          backgroundColor: Colors.red,
        ),
      );
    }
  }
  
  Future<void> _stopListening() async {
    await _playerController.stopListening();
    setState(() {});
  }
  
  @override
  void dispose() {
    _playerController.stopListening();
    _audioPlayer.dispose();
    super.dispose();
  }
  
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('راديو كارمش'),
        actions: [
          if (_isStreamLive)
            Container(
              padding: EdgeInsets.symmetric(horizontal: 12, vertical: 6),
              decoration: BoxDecoration(
                color: Colors.red,
                borderRadius: BorderRadius.circular(20),
              ),
              child: Row(
                children: [
                  Icon(Icons.radio, color: Colors.white, size: 16),
                  SizedBox(width: 4),
                  Text(
                    'بث مباشر',
                    style: TextStyle(color: Colors.white, fontSize: 12),
                  ),
                ],
              ),
            ),
        ],
      ),
      body: Column(
        children: [
          // UI للتحكم في البث
          ElevatedButton(
            onPressed: _playerController.isPlaying ? _stopListening : _startListening,
            child: Text(_playerController.isPlaying ? 'إيقاف' : 'تشغيل'),
          ),
          
          // حالة البث
          if (!_isStreamLive)
            Container(
              padding: EdgeInsets.all(16),
              margin: EdgeInsets.all(16),
              decoration: BoxDecoration(
                color: Colors.orange.shade100,
                borderRadius: BorderRadius.circular(8),
                border: Border.all(color: Colors.orange),
              ),
              child: Row(
                children: [
                  Icon(Icons.info_outline, color: Colors.orange),
                  SizedBox(width: 8),
                  Expanded(
                    child: Text(
                      'البث غير نشط حالياً. سيتم إيقاف الاستماع تلقائياً.',
                      style: TextStyle(color: Colors.orange.shade900),
                    ),
                  ),
                ],
              ),
            ),
        ],
      ),
    );
  }
}
```

## ⚠️ ملاحظات مهمة

### 1. Real-time Subscription مطلوب

**بدون Real-time subscription، الموبايل لن يعرف بإيقاف البث تلقائياً.**

### 2. التحقق من حالة البث قبل البدء

**قبل بدء الاستماع، تحقق من `is_live = true`:**

```dart
final stream = await getCurrentStream();
if (stream == null || !stream['is_live']) {
  // البث غير نشط
  return;
}
```

### 3. معالجة إيقاف البث

**عند إيقاف البث (`is_live = false`):**
- أوقف audio player فوراً
- أبلغ المستخدم بأن البث تم إيقافه
- أوقف جلسة الاستماع في قاعدة البيانات
- ألغِ Real-time subscriptions

### 4. إعادة التشغيل التلقائي (اختياري)

**يمكن إضافة منطق لإعادة التشغيل التلقائي عند تفعيل البث مرة أخرى:**

```dart
if (isLive && !isPlaying) {
  // البث تم تفعيله - يمكن إعادة التشغيل التلقائي
  await audioPlayer.play();
}
```
