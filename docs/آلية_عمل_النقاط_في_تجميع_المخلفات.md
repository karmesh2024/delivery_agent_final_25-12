## 🎯 نظرة عامة

### ما هي النقاط؟
النقاط هي نظام مكافآت يحصل عليه العملاء مقابل تسليم المخلفات القابلة لإعادة التدوير. كل كيلوجرام من المخلفات يعطي عدداً محدداً من النقاط حسب نوع الفئة الفرعية.

### المكونات الرئيسية:
1. **الفئات الفرعية** (`waste_sub_categories`): تحدد `points_per_kg` لكل فئة
2. **المنتجات** (`waste_data_admin`): تحتوي على الوزن والنقاط المحسوبة
3. **جلسات التجميع** (`waste_collection_sessions`): تجمع المخلفات من العميل
4. **عناصر التجميع** (`waste_collection_items`): كل عنصر في الجلسة مع نقاطه

---

## 🔄 السيناريو الكامل

### المرحلة 1: إعداد النظام

#### 1.1 إنشاء/تحديث الفئة الفرعية
```
المسؤول → إدارة الفئات → فئة فرعية جديدة
  ↓
تحديد points_per_kg (مثال: 10 نقاط لكل كيلو)
  ↓
حفظ في waste_sub_categories
```

**مثال:**
- فئة فرعية: "بلاستيك PET"
- `points_per_kg = 10`
- يعني: كل كيلوجرام بلاستيك PET = 10 نقاط

#### 1.2 إضافة منتج جديد
```
المسؤول → إدارة المنتجات → منتج جديد
  ↓
اختيار الفئة الفرعية
  ↓
إدخال الوزن (مثال: 0.5 كجم)
  ↓
النظام يحسب النقاط تلقائياً:
  calculated_points = weight × points_per_kg
  = 0.5 × 10 = 5 نقاط
```

---

### المرحلة 2: عملية التجميع

#### 2.1 العميل يطلب تجميع المخلفات
```
العميل → تطبيق الموبايل → طلب تجميع
  ↓
إنشاء delivery_order
  ↓
إنشاء waste_collection_session
  status = "pending"
```

#### 2.2 عامل التوصيل يبدأ الجلسة
```
عامل التوصيل → تطبيق التوصيل → بدء التجميع
  ↓
تحديث waste_collection_session:
  status = "in_progress"
  started_at = NOW()
```

#### 2.3 إضافة عناصر المخلفات
```
عامل التوصيل → وزن كل نوع من المخلفات
  ↓
لكل نوع:
  - اختيار المنتج (waste_data_id)
  - إدخال الوزن الفعلي (actual_weight)
  ↓
إنشاء waste_collection_item:
  - waste_data_id
  - actual_weight
  - حساب earned_points = actual_weight × points_per_kg
```

**مثال عملي:**
```
عنصر 1: بلاستيك PET
  - actual_weight = 2.5 كجم
  - points_per_kg = 10
  - earned_points = 2.5 × 10 = 25 نقطة

عنصر 2: ورق
  - actual_weight = 1.0 كجم
  - points_per_kg = 5
  - earned_points = 1.0 × 5 = 5 نقاط

عنصر 3: معادن
  - actual_weight = 0.5 كجم
  - points_per_kg = 15
  - earned_points = 0.5 × 15 = 7.5 نقاط (تُقرب إلى 8)
```

#### 2.4 إتمام الجلسة
```
عامل التوصيل → إنهاء التجميع
  ↓
تحديث waste_collection_session:
  status = "completed"
  completed_at = NOW()
  total_weight = SUM(actual_weight) من جميع العناصر
  total_amount = SUM(total_price) من جميع العناصر
  total_points = SUM(earned_points) من جميع العناصر
```

**مثال:**
```
total_weight = 2.5 + 1.0 + 0.5 = 4.0 كجم
total_points = 25 + 5 + 8 = 38 نقطة
```

#### 2.5 منح النقاط للعميل
```
عند إتمام الجلسة (status = "completed")
  ↓
تحديث رصيد نقاط العميل:
  profiles.loyalty_points += total_points
  ↓
تسجيل المعاملة (إن وجدت):
  points_transactions (إيداع)
```

## 🗄️ هيكل قاعدة البيانات

### 1. جدول `waste_sub_categories` (الفئات الفرعية)

```sql
CREATE TABLE waste_sub_categories (
    id                  BIGINT PRIMARY KEY,
    code                TEXT UNIQUE NOT NULL,
    name                TEXT NOT NULL,
    main_id             BIGINT REFERENCES waste_main_categories(id),
    points_per_kg       INTEGER DEFAULT 0,  -- ⭐ النقاط لكل كيلوجرام
    price               FLOAT,
    created_at          TIMESTAMPTZ,
    updated_at          TIMESTAMPTZ
);
```

**الحقول المهمة:**
- `points_per_kg`: عدد النقاط لكل كيلوجرام من هذا النوع

**مثال:**
```sql
INSERT INTO waste_sub_categories (id, code, name, points_per_kg)
VALUES (1, 'PET', 'بلاستيك PET', 10);
```

---

### 2. جدول `waste_data_admin` (المنتجات)

```sql
CREATE TABLE waste_data_admin (
    id                  UUID PRIMARY KEY,
    name                TEXT NOT NULL,
    subcategory_id      BIGINT REFERENCES waste_sub_categories(id),
    weight              FLOAT,                    -- الوزن بالكيلوجرام
    calculated_points   INTEGER DEFAULT 0,       -- ⭐ النقاط المحسوبة
    calculated_price    DECIMAL(10,2) DEFAULT 0,
    points_multiplier   DECIMAL(3,2) DEFAULT 1.00,
    created_at          TIMESTAMPTZ,
    updated_at          TIMESTAMPTZ
);
```

**الحقول المهمة:**
- `weight`: وزن المنتج بالكيلوجرام
- `calculated_points`: النقاط المحسوبة تلقائياً (weight × points_per_kg)
- `subcategory_id`: ربط بالفئة الفرعية للحصول على `points_per_kg`

---

### 3. جدول `waste_collection_sessions` (جلسات التجميع)

```sql
CREATE TABLE waste_collection_sessions (
    id                  UUID PRIMARY KEY,
    delivery_order_id   UUID NOT NULL,
    customer_id         UUID NOT NULL,
    delivery_boy_id     UUID NOT NULL,
    status              VARCHAR(20) DEFAULT 'pending',
    total_weight        DECIMAL(10,2) DEFAULT 0,   -- ⭐ إجمالي الوزن
    total_amount        DECIMAL(10,2) DEFAULT 0,    -- ⭐ إجمالي المبلغ
    total_points        DECIMAL(10,2) DEFAULT 0,    -- ⭐ إجمالي النقاط
    started_at          TIMESTAMPTZ,
    completed_at        TIMESTAMPTZ,
    created_at          TIMESTAMPTZ,
    updated_at          TIMESTAMPTZ
);
```

**الحقول المهمة:**
- `total_points`: إجمالي النقاط المكتسبة في هذه الجلسة
- `status`: حالة الجلسة (pending, in_progress, completed)

---

### 4. جدول `waste_collection_items` (عناصر التجميع)

```sql
CREATE TABLE waste_collection_items (
    id                  UUID PRIMARY KEY,
    session_id          UUID REFERENCES waste_collection_sessions(id),
    waste_data_id       UUID REFERENCES waste_data_admin(id),
    subcategory_id      UUID,
    name                VARCHAR(255),
    actual_weight       DECIMAL(10,3) NOT NULL,     -- ⭐ الوزن الفعلي المقاس
    unit_price          DECIMAL(10,2),
    total_price         DECIMAL(10,2),
    earned_points       DECIMAL(10,2) DEFAULT 0,    -- ⭐ النقاط المكتسبة لهذا العنصر
    created_at          TIMESTAMPTZ,
    updated_at          TIMESTAMPTZ
);
```

**الحقول المهمة:**
- `actual_weight`: الوزن الفعلي المقاس (قد يختلف عن وزن المنتج الافتراضي)
- `earned_points`: النقاط المكتسبة لهذا العنصر = `actual_weight × points_per_kg`

---

### 5. جدول `profiles` (ملفات العملاء)

```sql
CREATE TABLE profiles (

    updated_at          TIMESTAMPTZ

```

**الحقول المهمة:**
- `loyalty_points`: رصيد النقاط الإجمالي للعميل

## 🧮 آلية حساب النقاط

### الصيغة الأساسية

```
النقاط = الوزن (بالكيلوجرام) × points_per_kg
```

### الخطوات التفصيلية

#### 1. حساب النقاط للمنتج (`waste_data_admin`)

```sql
-- عند إنشاء/تحديث منتج
calculated_points = weight × (
    SELECT points_per_kg 
    FROM waste_sub_categories 
    WHERE id = waste_data_admin.subcategory_id
)
```

**مثال:**
```sql
-- منتج: زجاجة بلاستيك PET
weight = 0.5 كجم
subcategory_id = 1 (PET)
points_per_kg = 10

calculated_points = 0.5 × 10 = 5 نقاط
```

---

#### 2. حساب النقاط لعنصر التجميع (`waste_collection_items`)

```sql
-- عند إضافة عنصر في جلسة التجميع
earned_points = actual_weight × (
    SELECT points_per_kg 
    FROM waste_sub_categories 
    WHERE id = (
        SELECT subcategory_id 
        FROM waste_data_admin 
        WHERE id = waste_collection_items.waste_data_id
    )
)
```

**مثال:**
```sql
-- عنصر في جلسة التجميع
waste_data_id = 'uuid-123' (زجاجة PET)
actual_weight = 2.5 كجم (وزن فعلي مقاس)
subcategory_id = 1 (PET)
points_per_kg = 10

earned_points = 2.5 × 10 = 25 نقطة
```

---

#### 3. حساب إجمالي النقاط للجلسة (`waste_collection_sessions`)

```sql
-- عند إتمام الجلسة
total_points = SUM(earned_points) 
FROM waste_collection_items 
WHERE session_id = waste_collection_sessions.id
```

**مثال:**
```sql
-- جلسة تحتوي على 3 عناصر
عنصر 1: earned_points = 25
عنصر 2: earned_points = 5
عنصر 3: earned_points = 8

total_points = 25 + 5 + 8 = 38 نقطة
```

---

#### 4. تحديث رصيد العميل (`profiles`)

```sql
-- عند إتمام الجلسة (status = 'completed')
UPDATE profiles
SET loyalty_points = loyalty_points + (
    SELECT total_points 
    FROM waste_collection_sessions 
    WHERE id = 'session-uuid' 
    AND status = 'completed'
)
WHERE id = customer_id;
```

**مثال:**
```sql
-- العميل لديه 100 نقطة
-- الجلسة المكتسبة: 38 نقطة

loyalty_points = 100 + 38 = 138 نقطة
```

## ⚙️ التنفيذ التقني

### 1. Triggers والدوال

#### أ. حساب النقاط عند تحديث المنتج

```sql
-- دالة لحساب النقاط تلقائياً
CREATE OR REPLACE FUNCTION calculate_points_for_product()
RETURNS TRIGGER AS $$
DECLARE
    v_points_per_kg INTEGER;
BEGIN
    -- جلب points_per_kg من الفئة الفرعية
    SELECT points_per_kg INTO v_points_per_kg
    FROM waste_sub_categories
    WHERE id = NEW.subcategory_id;
    
    -- حساب النقاط
    IF v_points_per_kg IS NOT NULL AND NEW.weight IS NOT NULL THEN
        NEW.calculated_points := FLOOR(NEW.weight * v_points_per_kg);
    ELSE
        NEW.calculated_points := 0;
    END IF;
    
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

-- Trigger على INSERT/UPDATE
CREATE TRIGGER trigger_calculate_product_points
BEFORE INSERT OR UPDATE ON waste_data_admin
FOR EACH ROW
EXECUTE FUNCTION calculate_points_for_product();
```

---

#### ب. حساب النقاط عند إضافة عنصر التجميع

```sql
-- دالة لحساب النقاط لعنصر التجميع
CREATE OR REPLACE FUNCTION calculate_item_points()
RETURNS TRIGGER AS $$
DECLARE
    v_points_per_kg INTEGER;
    v_subcategory_id BIGINT;
BEGIN
    -- جلب subcategory_id من المنتج
    SELECT subcategory_id INTO v_subcategory_id
    FROM waste_data_admin
    WHERE id = NEW.waste_data_id;
    
    -- جلب points_per_kg من الفئة الفرعية
    SELECT points_per_kg INTO v_points_per_kg
    FROM waste_sub_categories
    WHERE id = v_subcategory_id;
    
    -- حساب النقاط
    IF v_points_per_kg IS NOT NULL AND NEW.actual_weight IS NOT NULL THEN
        NEW.earned_points := FLOOR(NEW.actual_weight * v_points_per_kg);
    ELSE
        NEW.earned_points := 0;
    END IF;
    
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

-- Trigger على INSERT/UPDATE
CREATE TRIGGER trigger_calculate_item_points
BEFORE INSERT OR UPDATE ON waste_collection_items
FOR EACH ROW
EXECUTE FUNCTION calculate_item_points();
```

#### ج. تحديث إجمالي النقاط للجلسة

```sql
-- دالة لتحديث إجمالي النقاط للجلسة
CREATE OR REPLACE FUNCTION update_session_totals()
RETURNS TRIGGER AS $$
DECLARE
    v_total_points DECIMAL(10,2);
    v_total_weight DECIMAL(10,2);
    v_total_amount DECIMAL(10,2);
BEGIN
    -- حساب الإجماليات من جميع العناصر
    SELECT 
        COALESCE(SUM(earned_points), 0),
        COALESCE(SUM(actual_weight), 0),
        COALESCE(SUM(total_price), 0)
    INTO v_total_points, v_total_weight, v_total_amount
    FROM waste_collection_items
    WHERE session_id = COALESCE(NEW.session_id, OLD.session_id);
    
    -- تحديث الجلسة
    UPDATE waste_collection_sessions
    SET 
        total_points = v_total_points,
        total_weight = v_total_weight,
        total_amount = v_total_amount,
        updated_at = NOW()
    WHERE id = COALESCE(NEW.session_id, OLD.session_id);
    
    RETURN COALESCE(NEW, OLD);
END;
$$ LANGUAGE plpgsql;

-- Trigger على INSERT/UPDATE/DELETE للعناصر
CREATE TRIGGER trigger_update_session_totals
AFTER INSERT OR UPDATE OR DELETE ON waste_collection_items
FOR EACH ROW
EXECUTE FUNCTION update_session_totals();
```

#### د. منح النقاط للعميل عند إتمام الجلسة

```sql
-- دالة لمنح النقاط عند إتمام الجلسة
CREATE OR REPLACE FUNCTION award_points_on_completion()
RETURNS TRIGGER AS $$
BEGIN
    -- فقط عند تغيير الحالة إلى "completed"
    IF NEW.status = 'completed' AND OLD.status != 'completed' THEN
        -- تحديث رصيد النقاط
        UPDATE profiles
        SET loyalty_points = loyalty_points + COALESCE(NEW.total_points, 0)
        WHERE id = NEW.customer_id;
        
        -- يمكن إضافة سجل في points_transactions هنا
        -- INSERT INTO points_transactions (...)
    END IF;
    
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

-- Trigger على UPDATE للجلسة
CREATE TRIGGER trigger_award_points
AFTER UPDATE ON waste_collection_sessions
FOR EACH ROW
EXECUTE FUNCTION award_points_on_completion();
```

### 2. الكود في التطبيق (TypeScript/React)

#### أ. حساب النقاط عند إنشاء منتج

```typescript
// في productService.ts
async createProduct(product: Product) {
  const { data: subcategory } = await supabase
    .from('waste_sub_categories')
    .select('points_per_kg')
    .eq('id', product.subcategory_id)
    .single();
  
  const calculatedPoints = Math.floor(
    product.weight * (subcategory?.points_per_kg || 0)
  );
  
  const { data, error } = await supabase
    .from('waste_data_admin')
    .insert({
      ...product,
      calculated_points: calculatedPoints,
    })
    .select()
    .single();
  
  return { data, error };
}
```

#### ب. إضافة عنصر في جلسة التجميع

```typescript
// في wasteCollectionService.ts
async addCollectionItem(item: {
  session_id: string;
  waste_data_id: string;
  actual_weight: number;
}) {
  // جلب معلومات المنتج
  const { data: product } = await supabase
    .from('waste_data_admin')
    .select('*, waste_sub_categories(points_per_kg)')
    .eq('id', item.waste_data_id)
    .single();
  
  // حساب النقاط
  const pointsPerKg = product.waste_sub_categories?.points_per_kg || 0;
  const earnedPoints = Math.floor(item.actual_weight * pointsPerKg);
  
  // إضافة العنصر
  const { data, error } = await supabase
    .from('waste_collection_items')
    .insert({
      ...item,
      earned_points: earnedPoints,
    })
    .select()
    .single();
  
  // الـ trigger سيحدث total_points في الجلسة تلقائياً
  
  return { data, error };
}
```

#### ج. إتمام الجلسة ومنح النقاط

```typescript
// في wasteCollectionService.ts
async completeSession(sessionId: string) {
  // تحديث حالة الجلسة
  const { data: session, error } = await supabase
    .from('waste_collection_sessions')
    .update({
      status: 'completed',
      completed_at: new Date().toISOString(),
    })
    .eq('id', sessionId)
    .select()
    .single();
  
  if (error) return { error };
  
  // الـ trigger سيحدث رصيد النقاط تلقائياً
  // لكن يمكننا التحقق:
  const { data: customer } = await supabase
    .from('profiles')
    .select('loyalty_points')
    .eq('id', session.customer_id)
    .single();
  
  return {
    data: {
      session,
      customer_points: customer?.loyalty_points,
    },
    error: null,
  };
}
```

## ✅ التحقق والصيانة

### 1. التحقق من صحة الحسابات

#### أ. التحقق من نقاط المنتج
```sql
-- التحقق من أن calculated_points صحيحة
SELECT 
    wda.id,
    wda.name,
    wda.weight,
    wsc.points_per_kg,
    wda.calculated_points,
    FLOOR(wda.weight * wsc.points_per_kg) as expected_points,
    CASE 
        WHEN wda.calculated_points = FLOOR(wda.weight * wsc.points_per_kg) 
        THEN '✓ صحيح' 
        ELSE '✗ خطأ' 
    END as status
FROM waste_data_admin wda
JOIN waste_sub_categories wsc ON wda.subcategory_id = wsc.id
WHERE wda.weight IS NOT NULL;
```

---

#### ب. التحقق من نقاط العناصر
```sql

SELECT 
    wci.id,
    wci.actual_weight,
    wsc.points_per_kg,
    wci.earned_points,
    FLOOR(wci.actual_weight * wsc.points_per_kg) as expected_points,
    CASE 
        WHEN wci.earned_points = FLOOR(wci.actual_weight * wsc.points_per_kg) 
        THEN '✓ صحيح' 
        ELSE '✗ خطأ' 
    END as status
FROM waste_collection_items wci
JOIN waste_data_admin wda ON wci.waste_data_id = wda.id
JOIN waste_sub_categories wsc ON wda.subcategory_id = wsc.id;
```

---

#### ج. التحقق من إجمالي النقاط في الجلسة
```sql
-- التحقق من أن total_points صحيحة
SELECT 
    wcs.id,
    wcs.total_points as session_total,
    COALESCE(SUM(wci.earned_points), 0) as calculated_total,
    CASE 
        WHEN wcs.total_points = COALESCE(SUM(wci.earned_points), 0) 
        THEN '✓ صحيح' 
        ELSE '✗ خطأ' 
    END as status
FROM waste_collection_sessions wcs
LEFT JOIN waste_collection_items wci ON wcs.id = wci.session_id
GROUP BY wcs.id, wcs.total_points;
```

---

### 2. إصلاح البيانات غير الصحيحة

#### أ. إعادة حساب نقاط المنتجات
```sql
-- تحديث calculated_points لجميع المنتجات
UPDATE waste_data_admin wda
SET calculated_points = FLOOR(
    wda.weight * COALESCE(wsc.points_per_kg, 0)
)
FROM waste_sub_categories wsc
WHERE wda.subcategory_id = wsc.id
AND wda.weight IS NOT NULL;
```

---

#### ب. إعادة حساب نقاط العناصر
```sql
-- تحديث earned_points لجميع العناصر
UPDATE waste_collection_items wci
SET earned_points = FLOOR(
    wci.actual_weight * COALESCE(wsc.points_per_kg, 0)
)
FROM waste_data_admin wda
JOIN waste_sub_categories wsc ON wda.subcategory_id = wsc.id
WHERE wci.waste_data_id = wda.id
AND wci.actual_weight IS NOT NULL;
```

---

#### ج. إعادة حساب إجمالي النقاط للجلسات
```sql
-- تحديث total_points لجميع الجلسات
UPDATE waste_collection_sessions wcs
SET total_points = COALESCE(
    (SELECT SUM(earned_points) 
     FROM waste_collection_items 
     WHERE session_id = wcs.id),
    0
);
```

### 3. مراقبة الأداء

#### أ. إحصائيات النقاط
```sql
-- إجمالي النقاط الممنوحة
SELECT 
    COUNT(*) as total_sessions,
    SUM(total_points) as total_points_awarded,
    AVG(total_points) as avg_points_per_session,
    MAX(total_points) as max_points_session,
    MIN(total_points) as min_points_session
FROM waste_collection_sessions
WHERE status = 'completed';
```

---

#### ب. النقاط حسب الفئة الفرعية
```sql
-- توزيع النقاط حسب نوع المخلفات
SELECT 
    wsc.name as subcategory_name,
    COUNT(*) as item_count,
    SUM(wci.earned_points) as total_points,
    AVG(wci.earned_points) as avg_points_per_item
FROM waste_collection_items wci
JOIN waste_data_admin wda ON wci.waste_data_id = wda.id
JOIN waste_sub_categories wsc ON wda.subcategory_id = wsc.id
GROUP BY wsc.id, wsc.name
ORDER BY total_points DESC;
```

## 📝 ملخص

### التدفق الكامل:

```
1. إعداد النظام
   ├─ إنشاء فئة فرعية (points_per_kg)
   └─ إضافة منتج (weight → calculated_points)

2. عملية التجميع
   ├─ إنشاء جلسة (pending)
   ├─ بدء الجلسة (in_progress)
   ├─ إضافة عناصر (actual_weight → earned_points)
   ├─ إتمام الجلسة (completed)
   └─ منح النقاط (total_points → loyalty_points)

3. التحقق
   ├─ التحقق من صحة الحسابات
   ├─ إصلاح البيانات غير الصحيحة
   └─ مراقبة الأداء
```

### الصيغ الأساسية:

```
1. نقاط المنتج:
   calculated_points = weight × points_per_kg

2. نقاط العنصر:
   earned_points = actual_weight × points_per_kg

3. إجمالي النقاط:
   total_points = SUM(earned_points)

4. رصيد العميل:
   loyalty_points += total_points (عند إتمام الجلسة)
```
