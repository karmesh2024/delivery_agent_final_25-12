## 📋 مصادر الاستلام المدعومة

### 1. **الدليفري بوي** 🚚
```typescript
{
  source: 'delivery_boy',
  delivery_agent_id: string,
  collection_session_id?: string,
  warehouse_id: number
}
```

### 2. **الموردين** 📦
```typescript
{
  source: 'supplier',
  supplier_id: number,
  supplier_invoice_id?: string, // UUID
  warehouse_id: number
}
```

### 3. **الوكلاء** 👤
```typescript
{
  source: 'agent',
  agent_id: string, // UUID
  warehouse_id: number
}
```

### 4. **مباشر** 📥
```typescript
{
  source: 'direct',
  warehouse_id: number
}
```
