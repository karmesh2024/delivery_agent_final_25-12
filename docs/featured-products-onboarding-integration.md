# ميزة المنتجات المميزة في الاقتراحات الذكية

## نظرة عامة

تم إضافة ميزة "المنتجات المميزة" (Featured Products) التي تسمح للأدمن بتمييز منتجات معينة لتظهر في المساعد الذكي للمستخدمين الجدد في تطبيق الموبايل.

## التغييرات المنفذة

### 1. قاعدة البيانات

تم إضافة حقل جديد في جدول `waste_data_admin`:
- **اسم الحقل**: `is_onboarding_featured`
- **النوع**: `BOOLEAN`
- **القيمة الافتراضية**: `false`

**Migration**: `prisma/migrations/20260124_add_is_onboarding_featured/migration.sql`

### 2. واجهة المستخدم

#### أ. صفحة قائمة المنتجات (`ProductsListPage.tsx`)
- تم إضافة عمود جديد في الجدول: "إظهار في الاقتراحات الذكية"
- تم إضافة مكون Switch (Toggle) لكل منتج للتحكم السريع في حالة التمييز
- عند التبديل، يتم تحديث الحقل مباشرة في قاعدة البيانات

#### ب. نموذج المنتج (`ProductForm.tsx`)
- تم إضافة Switch في نموذج إضافة/تعديل المنتج
- يمكن للأدمن تحديد ما إذا كان المنتج سيظهر في الاقتراحات الذكية عند إنشاء أو تعديل المنتج

### 3. الخدمات

تم تحديث `productService.ts`:
- تم إضافة الحقل `is_onboarding_featured` في واجهة `Product`
- الخدمة تدعم الآن قراءة وكتابة هذا الحقل

## كيفية الاستخدام

### للأدمن في لوحة التحكم:

1. **من صفحة قائمة المنتجات**:
   - انتقل إلى صفحة المنتجات
   - ستجد عمود "إظهار في الاقتراحات الذكية" في الجدول
   - استخدم Switch بجانب كل منتج لتفعيل/إلغاء التفعيل

2. **من نموذج إضافة/تعديل المنتج**:
   - عند إضافة منتج جديد أو تعديل منتج موجود
   - ستجد خيار "إظهار في الاقتراحات الذكية" في النموذج
   - قم بتفعيل Switch لحفظ المنتج كمميز

### لتطبيق الموبايل:

التطبيق يستخدم الاستعلام التالي لجلب المنتجات المميزة:

```javascript
const { data } = await supabase
  .from('waste_data_admin')
  .select('*')
  .eq('is_onboarding_featured', true)
  .filter('waste_sub_category.main_id', 'in', `(${selectedIds})`);
```

## أفضل الممارسات

1. **العدد المثالي**: يفضل تمييز 3-5 منتجات كحد أقصى لكل فئة رئيسية
2. **التنوع**: اختر المنتجات الأكثر شيوعاً في المنازل (كرتون، عبوات مياه، علب كانز)
3. **جودة الصور**: تأكد من أن الصور المرفوعة للمنتجات المميزة ذات جودة جيدة (PNG شفافة يفضل)
