## 🎯 **نظرة عامة**

### **ما هو المشروع؟**

نظام أتمتة ذكي مدعوم بـ **Google Gemini 1.5 Flash** و**محرك نفسي متقدم** لإنشاء محتوى سوشيال ميديا احترافي عن منطقة **محرم بك** في الإسكندرية بشكل تلقائي، مستمر، ومجاني 100%.

```
📰 جمع الأخبار → 🧠 تحليل نفسي → ✍️ كتابة ذكية → 🎨 توليد صور → 💾 حفظ في Supabase → 📱 نشر
```

### **الهدف:**

إنشاء **45 بوست يومياً** (15 خبر × 3 أساليب) مع **تحليل نفسي كامل** جاهزة للنشر على Facebook/Instagram/Twitter بدون تدخل يدوي، مع:

- ✅ **جودة عالية** - محتوى احترافي
- ✅ **ذكاء نفسي** - يفهم جمهور محرم بك
- ✅ **تحليل المشاعر** - positive/negative/neutral
- ✅ **محفزات نفسية** - belonging, pride, hope
- ✅ **تكييف زمني** - حسب الوقت من اليوم
- ✅ **فهم ثقافي** - مصري/سكندري أصيل
- ✅ **صور AI** - جذابة ومناسبة

### **الميزة الفريدة: المحرك النفسي 🧠**

هذا ليس مجرد نظام أتمتة - إنه **منصة ذكاء نفسي** تفهم:
- 💭 مشاعر الجمهور (فرح، أمل، قلق، فخر)
- 🎯 المحفزات النفسية (الانتماء، الأمان، الفضول)
- 🕐 الحالة النفسية حسب الوقت (صباح/مساء/ليل)
- 🇪🇬 السياق الثقافي المصري/السكندري

### **المرحلة 4: توليد الصور (20 ثانية/صورة)**

#### **ماذا يحدث؟**

النظام يستخدم **Stable Diffusion XL** عبر Hugging Face لتوليد صورة مناسبة:

```javascript
const imagePrompt = "Modern medical center in Alexandria Egypt, Moharram Bek neighborhood, clean and bright, healthcare facility, realistic photography, high quality";

// إرسال إلى Stable Diffusion XL
const imageResponse = await axios({
  method: "POST",
  url: "https://api-inference.huggingface.co/models/stabilityai/stable-diffusion-xl-base-1.0",
  headers: {
    "Authorization": `Bearer ${HUGGINGFACE_TOKEN}`
  },
  data: {
    inputs: imagePrompt,
    options: { wait_for_model: true }
  },
  responseType: 'arraybuffer'
});

// تحويل إلى base64
const base64Image = Buffer.from(imageResponse.data).toString('base64');
const imageUrl = `data:image/png;base64,${base64Image}`;
```

**النتيجة:** صورة احترافية بدقة 1024×1024

✅ **نعم 100%!** جميع الخدمات المستخدمة لها free tiers سخية. لا تحتاج بطاقة ائتمان أبداً.

### **4. لماذا Gemini وليس Qwen؟**

✅ **تحليل نفسي متقدم** (يفهم السياق المصري بعمق)  
✅ **JSON Mode مدمج** (أقل أخطاء)

### **6. ماذا لو نفذت حصة Gemini (1,500/يوم)؟**
📊 أنت تستخدم **3% فقط** (45/يوم). لن تنفذ أبداً!
