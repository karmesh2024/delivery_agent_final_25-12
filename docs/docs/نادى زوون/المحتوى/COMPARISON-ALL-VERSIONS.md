### و) التحليلات والإحصائيات

#### بدون المحرك النفسي:
```sql
-- فقط عد المنشورات
SELECT COUNT(*) FROM posts;
```

#### مع المحرك النفسي 🧠:
```sql
-- تحليل نفسي شامل
SELECT 
  sentiment,
  COUNT(*) as count,
  STRING_AGG(DISTINCT emotions, ', ') as emotions,
  STRING_AGG(DISTINCT psychological_triggers, ', ') as triggers
FROM posts
WHERE created_at >= NOW() - INTERVAL '7 days'
GROUP BY sentiment;

-- النتيجة:
| sentiment | count | emotions | triggers |
|-----------|-------|----------|----------|
| positive  | 30    | فرح، أمل، فخر | belonging, pride, hope |
| negative  | 5     | قلق، تعاطف | safety, empathy |
| neutral   | 10    | اهتمام | curiosity |
```

**الفائدة:**
- معرفة نسبة الأخبار الإيجابية/السلبية
- فهم المشاعر السائدة في الحي
- تحسين الاستراتيجية بناءً على البيانات

## 💡 الحالات الاستخدامية الجديدة

### بفضل المحرك النفسي، أصبح يمكنك:

1. **فهم مزاج الحي** 📈
   - كم نسبة الأخبار الإيجابية هذا الأسبوع؟
   - ما المشاعر السائدة؟

2. **تحسين المحتوى** 🎯
   - أي أسلوب يحقق تفاعل أكثر؟
   - أي محفزات نفسية تعمل بشكل أفضل؟

3. **التخطيط الاستراتيجي** 🗓️
   - في أوقات الأخبار السلبية → نشر محتوى تحفيزي أكثر
   - في رمضان → استهداف التضامن والخير

4. **A/B Testing نفسي** 🧪
   - اختبار محفزات مختلفة
   - قياس الأداء لكل محفز

5. **بناء علاقة عاطفية** ❤️
   - المحتوى لم يعد "معلومات فقط"
   - أصبح يستهدف مشاعر وانتماء
