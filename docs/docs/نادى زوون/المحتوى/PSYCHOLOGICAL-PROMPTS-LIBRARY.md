## Prompts حسب نوع الخبر

### 1️⃣ أخبار إيجابية (Positive News)

```javascript
const positiveNewsPrompt = `أنت خبير في التسويق العاطفي الإيجابي.

الخبر الإيجابي: ${newsTitle}

🎯 التحليل النفسي المطلوب:
- sentiment: "positive"
- emotions: فرح، أمل، فخر، حماس
- triggers: pride (الفخر بالحي), hope (الأمل في المستقبل), belonging (الانتماء)
- tone: celebratory (احتفالي)

📝 أسلوب الكتابة:
- استخدم emojis إيجابية (🎉 ✨ 💚 🌟)
- ابدأ بكلمات مثل: "أخيراً!", "يا فرحة!", "بشرى سارة!"
- أنهي بدعوة للاحتفال أو المشاركة
- ركز على الأثر الإيجابي على الحي

مثال للنبرة:
"🎉 يا فرحة أهل محرم بك! مستشفى جديد بيفتح في حينا! 
الحي بتاعنا بيتطور كل يوم... مين هيكون أول واحد يزوره؟ 💚"

الآن اكتب 3 منشورات (JSON)...`;
```

📝 أسلوب الكتابة:
- استخدم لغة محترمة ومتعاطفة

- emojis: محدودة وجادة (🙏 💔 ⚠️)

## المحفزات النفسية (Psychological Triggers)

### 1. Belonging (الانتماء)

```
الكلمات المفتاحية: "حينا", "بتاعنا", "أهل محرم بك", "عيلتنا"
الهدف: جعل القارئ يشعر بالارتباط بالحي
مثال: "محرم بك... الحي بتاعنا اللي بنفتخر بيه 💚"
```

### 2. Pride (الفخر)

```
الكلمات المفتاحية: "فخورين", "عريق", "تاريخ", "أصالة"
الهدف: إثارة الفخر بالانتماء
مثال: "محرم بك العريق... حي له تاريخ وأصالة 🏛️"
```

### 3. Hope (الأمل)

```
الكلمات المفتاحية: "المستقبل", "تطوير", "نحو الأفضل", "واعد"
الهدف: خلق شعور بالتفاؤل
مثال: "محرم بك ماشي نحو مستقبل أفضل 🚀"
```

### 4. Safety (الأمان)

```
الكلمات المفتاحية: "آمن", "نطمن", "نحمي", "ناخد بالنا"
الهدف: طمأنة الجمهور
مثال: "خلونا ناخد بالنا من بعض عشان محرم بك تفضل آمنة 🛡️"
```

### 5. Curiosity (الفضول)

```
الكلمات المفتاحية: "إيه رأيكم؟", "حد عنده خبر؟", "تعرفوا؟"
الهدف: إثارة الفضول والتفاعل
مثال: "حاجة جديدة في محرم بك! حد سمع عنها؟ 🤔"
```

### 6. FOMO (الخوف من الفوات)

```
الكلمات المفتاحية: "ما تفوتكمش", "آخر فرصة", "محدود"
الهدف: خلق إلحاح للتفاعل
مثال: "عرض محدود في محرم بك! ما تفوتوش الفرصة! ⏰"
```

## 💡 نصائح لتحسين الـ Prompts

### 1. كن محدداً في السياق:
❌ سيء: "اكتب منشور عن الخبر"  
✅ جيد: "اكتب منشور يستهدف الفخر والانتماء، بأسلوب احتفالي، للجمهور السكندري"

### 2. اذكر المحفزات النفسية بوضوح:
```
المحفزات المطلوبة: belonging (الانتماء), pride (الفخر), hope (الأمل)
```

### 3. حدد الطول المطلوب:
```
- Formal: 150-180 حرف
- Casual: 120-150 حرف
- Interactive: 130-160 حرف + سؤال
```

### 4. اطلب JSON صريح:
```
responseMimeType: "application/json"
```

### 5. أعطي أمثلة في الـ Prompt (Few-Shot):
```
مثال على الأسلوب المطلوب:
"🏡 في محرم بك، الناس دايمًا متعاونة. الحي بتاعنا أحلى حي! #محرم_بك"
```
