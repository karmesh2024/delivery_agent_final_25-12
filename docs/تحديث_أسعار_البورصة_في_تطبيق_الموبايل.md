## الحل: إضافة Supabase Realtime Subscription

### للمطورين في تطبيق الموبايل (Flutter/React Native):

#### الطريقة 1: استخدام Supabase Realtime (مُوصى به)

```dart
// Flutter Example
import 'package:supabase_flutter/supabase_flutter.dart';

class StockExchangeService {
  final supabase = Supabase.instance.client;
  
  // الاشتراك في التحديثات الفورية لأسعار البورصة
  RealtimeChannel? priceChannel;
  
  void subscribeToPriceUpdates(Function(Map<String, dynamic>) onPriceUpdate) {
    // إنشاء channel للاشتراك في التحديثات
    priceChannel = supabase
        .channel('stock_exchange_prices')
        .onPostgresChanges(
          event: PostgresChangeEvent.update,
          schema: 'public',
          table: 'stock_exchange',
          callback: (payload) {
            // عند تحديث أي سعر في stock_exchange
            final updatedPrice = payload.newRecord;
            onPriceUpdate({
              'id': updatedPrice['id'],
              'product_id': updatedPrice['product_id'],
              'buy_price': updatedPrice['buy_price'],
              'last_actual_purchase_price': updatedPrice['last_actual_purchase_price'],
              'last_update': updatedPrice['last_update'],
            });
          },
        )
        .subscribe();
  }
  
  void unsubscribe() {
    priceChannel?.unsubscribe();
    supabase.removeChannel(priceChannel);
  }
}
```

#### استخدامه في الشاشة:

```dart
class WastePricesScreen extends StatefulWidget {
  @override
  _WastePricesScreenState createState() => _WastePricesScreenState();
}

class _WastePricesScreenState extends State<WastePricesScreen> {
  final stockExchangeService = StockExchangeService();
  List<Map<String, dynamic>> prices = [];
  
  @override
  void initState() {
    super.initState();
    // جلب الأسعار الأولية
    fetchInitialPrices();
    
    // الاشتراك في التحديثات الفورية
    stockExchangeService.subscribeToPriceUpdates((updatedPrice) {
      setState(() {
        // تحديث السعر في القائمة
        final index = prices.indexWhere((p) => p['id'] == updatedPrice['id']);
        if (index != -1) {
          prices[index] = updatedPrice;
        } else {
          prices.add(updatedPrice);
        }
      });
      
      // إظهار إشعار للمستخدم
      showNotification('تم تحديث سعر ${updatedPrice['name']}');
    });
  }
  
  Future<void> fetchInitialPrices() async {
    // جلب الأسعار الأولية من API
    final response = await http.get(
      Uri.parse('https://your-api.com/api/exchange/prices'),
    );
    final data = json.decode(response.body);
    setState(() {
      prices = List<Map<String, dynamic>>.from(data);
    });
  }
  
  @override
  void dispose() {
    stockExchangeService.unsubscribe();
    super.dispose();
  }
  
  @override
  Widget build(BuildContext context) {
    return ListView.builder(
      itemCount: prices.length,
      itemBuilder: (context, index) {
        final price = prices[index];
        return PriceCard(
          name: price['name'],
          currentPrice: price['buy_price'],
          lastPurchasePrice: price['last_actual_purchase_price'],
          change: calculateChange(
            price['buy_price'],
            price['last_actual_purchase_price'],
          ),
        );
      },
    );
  }
}
```

### الطريقة 2: Polling (بديل بسيط)

إذا لم يكن Supabase Realtime متاحاً، يمكن استخدام polling:

```dart
class PricePollingService {
  Timer? _timer;
  
  void startPolling(Function() onUpdate) {
    // تحديث كل 30 ثانية
    _timer = Timer.periodic(Duration(seconds: 30), (timer) {
      onUpdate();
    });
  }
  
  void stopPolling() {
    _timer?.cancel();
  }
}
```

## تفعيل Realtime في Supabase

### 1. تفعيل Realtime لجدول `stock_exchange`:

```sql
-- تفعيل Realtime لجدول stock_exchange
ALTER PUBLICATION supabase_realtime ADD TABLE stock_exchange;
```

### 2. التأكد من وجود RLS policies:

```sql
-- التأكد من أن المستخدمين يمكنهم قراءة stock_exchange
CREATE POLICY "Allow public read access to stock_exchange"
ON stock_exchange
FOR SELECT
USING (true);
```

## API Endpoint المحسّن

### تحديث `/api/exchange/prices` لإضافة معلومات إضافية:

```typescript
// src/app/api/exchange/prices/route.ts
export async function GET() {
    const { data, error } = await supabase
        .from("stock_exchange")
        .select(`
            id,
            product_id,
            buy_price,
            last_actual_purchase_price,
            price_change_percentage,
            demand_level,
            last_update,
            product:waste_data_admin(name, price_unit)
        `)
        .order("buy_price", { ascending: false });

    const formattedData = data?.map((item: any) => ({
        id: item.id,
        product_id: item.product_id,
        name: item.product?.name || "Unknown",
        unit: item.product?.price_unit || "Kg",
        current_price: item.buy_price,
        last_purchase_price: item.last_actual_purchase_price,
        change: item.last_actual_purchase_price 
            ? ((item.buy_price - item.last_actual_purchase_price) / item.last_actual_purchase_price * 100)
            : item.price_change_percentage,
        demand: item.demand_level,
        lastUpdated: item.last_update,
    })) || [];

    return NextResponse.json(formattedData);
}
```

## المزايا

### مع Realtime:
- ✅ تحديث فوري عند تغيير السعر
- ✅ لا حاجة لـ polling (توفير في البطارية والبيانات)
- ✅ تجربة مستخدم أفضل
- ✅ إشعارات فورية للمستخدم

### مع Polling:
- ✅ سهل التطبيق
- ✅ لا يحتاج إعدادات إضافية
- ❌ استهلاك أكبر للبطارية والبيانات
- ❌ تأخير في التحديث (حسب فترة polling)
