# نظام الإدارة العليا كمركز التحكم - دليل شامل

## 🏢 نظرة عامة على النظام الجديد

تم تطوير نظام إدارة المخازن لتحويل **الإدارة العليا** من مجرد مخزن إلى **نقطة العمل الأساسية** التي تدير وتنظم جميع المخازن والقطاعات والكتالوج.

## 🎯 المفهوم الجديد

### الإدارة العليا كمركز التحكم:
- **ليست مخزن** - بل هي إدارة عليا تدير النظام
- **تدير القطاعات** - تحدد القطاعات المتاحة لكل مستوى
- **تنشئ الكتالوج** - تضع الكتالوج الأساسي للمنتجات والمخلفات
- **تحدد الصلاحيات** - تحدد ما يمكن لكل مخزن إدارته

## 📊 الهيكل الجديد

### 1. الإدارة العليا (Admin Level)
- **الوظيفة**: مركز التحكم الرئيسي
- **المميزات**:
  - إدارة جميع القطاعات
  - إنشاء الكتالوج الأساسي
  - تحديد الصلاحيات لكل مستوى
  - مراقبة النظام العام

### 2. المخازن الرئيسية (Country Level)
- **الوظيفة**: إدارة المخازن على مستوى الدولة
- **الصلاحيات**: 
  - اختيار القطاعات من القائمة المعتمدة
  - إدارة مخازن المدن التابعة
  - تطبيق السياسات العامة

### 3. مخازن المدن (City Level)
- **الوظيفة**: إدارة المخازن على مستوى المدينة
- **الصلاحيات**:
  - اختيار القطاعات المناسبة للمدينة
  - إدارة مخازن المناطق التابعة
  - تنفيذ العمليات المحلية

### 4. مخازن المناطق (District Level)
- **الوظيفة**: إدارة المخازن على مستوى المنطقة
- **الصلاحيات**:
  - اختيار القطاعات المناسبة للمنطقة
  - تنفيذ العمليات اليومية
  - إدارة المخزون المحلي

## 🛠️ الملفات الجديدة

### 1. إدارة القطاعات والكتالوج
- **`src/app/warehouse-management/admin-settings/sectors/page.tsx`**: صفحة إدارة القطاعات والكتالوج
- **`warehouse_sectors_catalog_schema.sql`**: مخطط قاعدة البيانات للقطاعات والكتالوج

### 2. تحديثات النظام
- **`src/domains/warehouse-management/services/warehouseService.ts`**: إضافة دوال إدارة القطاعات
- **`src/app/warehouse-management/warehouses/new/page.tsx`**: تحديث نموذج إنشاء المخزن

## 🎯 المميزات الجديدة

### 1. إدارة القطاعات من الإدارة العليا
- **الوصول**: `/warehouse-management/admin-settings/sectors`
- **المميزات**:
  - إضافة قطاعات جديدة
  - تحديد المستويات المسموحة لكل قطاع
  - إدارة فئات المنتجات والمخلفات
  - ألوان مميزة للقطاعات

### 2. نظام اختيار القطاعات عند إنشاء المخزن
- **اختيار القطاعات**: عند إنشاء مخزن جديد، يجب اختيار القطاعات
- **تصفية حسب المستوى**: القطاعات المتاحة تختلف حسب مستوى المخزن
- **عرض بصري**: ألوان وأيقونات مميزة للقطاعات

### 3. الكتالوج الأساسي
- **فئات المنتجات**: إدارة فئات المنتجات الأساسية
- **فئات المخلفات**: إدارة فئات المخلفات الأساسية
- **ربط بالقطاعات**: كل فئة مرتبطة بقطاع معين

## 🔧 قاعدة البيانات الجديدة

### الجداول الجديدة:
```sql
-- القطاعات
warehouse_sectors
- id, name, description, code, color
- warehouse_levels (المستويات المسموحة)
- is_active, created_at, updated_at

-- فئات المنتجات
product_categories
- id, name, description, sector_id
- is_active, created_at, updated_at

-- فئات المخلفات
waste_categories
- id, name, description, sector_id
- is_active, created_at, updated_at

-- المنتجات الأساسية
base_products
- id, name, description, category_id
- unit, base_price, is_active

-- المخلفات الأساسية
base_waste
- id, name, description, category_id
- unit, disposal_cost, is_active

-- ربط المخازن بالقطاعات
warehouse_sector_assignments
- warehouse_id, sector_id, is_primary
- capacity_percentage, assigned_at
```

## 🚀 كيفية الاستخدام

### 1. إعداد الإدارة العليا
1. انتقل إلى `/warehouse-management/admin-settings`
2. قم بتكوين الإعدادات العامة
3. انتقل إلى `/warehouse-management/admin-settings/sectors`
4. أضف القطاعات المطلوبة
5. أضف فئات المنتجات والمخلفات

### 2. إنشاء المخازن مع القطاعات
1. انتقل إلى `/warehouse-management/warehouses/new`
2. اختر المستوى الهرمي (رئيسي، مدينة، منطقة)
3. اختر القطاعات المناسبة للمستوى
4. أكمل باقي بيانات المخزن
5. احفظ المخزن

### 3. إدارة القطاعات
1. انتقل إلى `/warehouse-management/admin-settings/sectors`
2. أضف قطاعات جديدة حسب الحاجة
3. حدد المستويات المسموحة لكل قطاع
4. أضف فئات المنتجات والمخلفات
5. ربط الفئات بالقطاعات المناسبة
