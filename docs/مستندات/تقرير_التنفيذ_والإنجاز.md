### 2. تحسين الأداء (Performance) - **متبقي 25%**

#### ❌ لم يتم: إنشاء دالة RPC لـ `getOrderStats`

- **المطلوب:** كتابة دالة SQL في Supabase تجلب كل الإحصائيات في استعلام واحد
- **الوضع الحالي:** الدالة تنفذ 5-6 استعلامات منفصلة
- **التحسين المتوقع:** تقليل عدد الاستعلامات من 6 إلى 1
- **الأولوية:** متوسطة

**مثال على الكود المطلوب:**

```sql
CREATE OR REPLACE FUNCTION get_order_stats()
RETURNS JSON AS $$
DECLARE
  result JSON;
BEGIN
  SELECT json_build_object(
    'total', (SELECT COUNT(*) FROM delivery_orders),
    'pending', (SELECT COUNT(*) FROM delivery_orders WHERE status = 'pending'),
    'in_progress', (SELECT COUNT(*) FROM delivery_orders WHERE status = 'inReceipt'),
    'delivered', (SELECT COUNT(*) FROM delivery_orders WHERE status = 'completed'),
    'canceled', (SELECT COUNT(*) FROM delivery_orders WHERE status = 'canceled')
  ) INTO result;
  
  RETURN result;
END;
$$ LANGUAGE plpgsql;
```
