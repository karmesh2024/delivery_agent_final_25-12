# دليل الوصول إلى صفحة إدارة القطاعات

## 🎯 الموقع المطلوب: `warehouse-management/admin-settings/sectors`

### الطريق الأول: من الصفحة الرئيسية

#### 1. **الصفحة الرئيسية لإدارة المخازن**
- انتقل إلى: `http://localhost:3000/warehouse-management`
- ابحث عن البطاقة المعنونة: **"الهيكل الهرمي للمخازن"**

#### 2. **البطاقة المطلوبة**
```
الهيكل الهرمي للمخازن
├── الوصف: "إدارة المخازن الهرمية (مخزن أم - مخزن مدينة - مخزن منطقة) وربطها بالقطاعات المختلفة."
├── زر: "إدارة الهيكل الهرمي"
└── زر: "إعدادات الإدارة العليا" ← اضغط هنا
```

#### 3. **الانتقال إلى الإعدادات**
- اضغط على زر **"إعدادات الإدارة العليا"**
- ستنتقل إلى: `/warehouse-management/admin-settings`

#### 4. **الوصول إلى القطاعات**
- في صفحة الإعدادات، ستجد زر **"إدارة القطاعات"** في الأعلى
- اضغط على زر **"إدارة القطاعات"**
- ستنتقل إلى: `/warehouse-management/admin-settings/sectors`

### الطريق الثاني: مباشرة

#### 1. **الرابط المباشر**
```
http://localhost:3000/warehouse-management/admin-settings/sectors
```

### الطريق الثالث: من الهيكل الهرمي

#### 1. **صفحة الهيكل الهرمي**
- انتقل إلى: `/warehouse-management/hierarchy`
- ابحث عن زر **"إدارة عليا"** في الأعلى
- اضغط على زر **"إدارة عليا"**

## 🎨 ما ستجده في صفحة القطاعات

### التبويبات المتاحة:
1. **القطاعات** - إدارة القطاعات الأساسية
2. **فئات المنتجات** - إدارة فئات المنتجات
3. **فئات المخلفات** - إدارة فئات المخلفات

### المميزات:
- ✅ إضافة قطاعات جديدة
- ✅ تحديد المستويات المسموحة لكل قطاع
- ✅ إدارة فئات المنتجات والمخلفات
- ✅ ألوان مميزة للقطاعات
- ✅ ربط الفئات بالقطاعات

## 🔧 إذا لم تجد الأزرار

### 1. **تأكد من تحديث الصفحة**
```bash
# إعادة تشغيل التطبيق
npm run dev
```

### 2. **تأكد من الملفات المحدثة**
- ✅ `src/app/warehouse-management/page.tsx` - الصفحة الرئيسية
- ✅ `src/app/warehouse-management/admin-settings/page.tsx` - صفحة الإعدادات
- ✅ `src/app/warehouse-management/admin-settings/sectors/page.tsx` - صفحة القطاعات

### 3. **تحقق من الروابط**
```typescript
// في الصفحة الرئيسية
<Link href="/warehouse-management/admin-settings">
  <Button variant="outline" className="w-full">إعدادات الإدارة العليا</Button>
</Link>

// في صفحة الإعدادات
<Link href="/warehouse-management/admin-settings/sectors">
  <Button variant="outline" className="flex items-center gap-2">
    <FiLayers className="w-4 h-4" />

```

## 🎉 النتيجة النهائية

بعد اتباع هذه الخطوات، ستجد نفسك في صفحة إدارة القطاعات حيث يمكنك:

1. **إضافة قطاعات جديدة** مع تحديد المستويات المسموحة
2. **إدارة فئات المنتجات** وربطها بالقطاعات
3. **إدارة فئات المخلفات** وربطها بالقطاعات
4. **تخصيص ألوان القطاعات** لسهولة التمييز
5. **إعداد الكتالوج الأساسي** للمنتجات والمخلفات
