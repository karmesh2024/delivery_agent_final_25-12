# 🏗️ دليل استخدام الهيكل الهرمي للمخازن

## 🎯 نظرة عامة

تم تطبيق الهيكل الهرمي للمخازن بنجاح! الآن يمكنك إدارة المخازن على ثلاثة مستويات:

### 📊 **المستويات الهرمية:**
1. **مخزن أم (Country Level)** - على مستوى الدولة
2. **مخزن مدينة (City Level)** - على مستوى المدينة  
3. **مخزن منطقة (District Level)** - على مستوى المنطقة

### 🔗 **القطاعات المدعومة:**
- **صناعي** (INDUSTRIAL) - النفايات الصناعية
- **تجاري** (COMMERCIAL) - النفايات التجارية
- **سياحي** (TOURISM) - النفايات السياحية
- **طبي** (MEDICAL) - النفايات الطبية
- **زراعي** (AGRICULTURAL) - النفايات الزراعية
- **إداري** (ADMINISTRATIVE) - النفايات الإدارية
- **منزلي** (HOUSEHOLD) - النفايات المنزلية

## 🚀 **كيفية الاستخدام**

### **1. الوصول للهيكل الهرمي**
```
http://localhost:3000/warehouse-management/hierarchy
```

### **2. إنشاء مخزن جديد**

#### **أ) إنشاء مخزن رئيسي (مستوى الدولة):**
1. انتقل إلى تبويب "شجرة المخازن"
2. اضغط على "إضافة مخزن رئيسي"
3. املأ البيانات:
   - **الاسم**: مخزن شمال الدلتا الرئيسي
   - **الموقع**: مصر - شمال الدلتا
   - **المستوى**: مخزن أم
   - **كود الدولة**: EGN

#### **ب) إنشاء مخزن مدينة:**
1. اختر المخزن الرئيسي من الشجرة
2. اضغط على زر "+" بجانب المخزن
3. املأ البيانات:
   - **الاسم**: مخزن الإسكندرية الرئيسي
   - **الموقع**: الإسكندرية - مصر
   - **المستوى**: مخزن مدينة
   - **المخزن الأب**: مخزن شمال الدلتا الرئيسي

#### **ج) إنشاء مخزن منطقة:**
1. اختر مخزن المدينة من الشجرة
2. اضغط على زر "+" بجانب المخزن
3. املأ البيانات:
   - **الاسم**: مخزن محرم بك شرق
   - **الموقع**: محرم بك - الإسكندرية
   - **المستوى**: مخزن منطقة
   - **المخزن الأب**: مخزن الإسكندرية الرئيسي

### **3. إدارة القطاعات**

#### **ربط قطاعات بالمخزن:**
1. اختر مخزن من الشجرة
2. انتقل إلى تبويب "إدارة القطاعات"
3. أضف القطاعات المطلوبة:
   - **القطاع الأساسي**: اختر القطاع الرئيسي للمخزن
   - **نسبة السعة**: حدد النسبة المئوية لكل قطاع
   - **القطاعات الثانوية**: أضف قطاعات إضافية

#### **مثال على ربط القطاعات:**
```
مخزن شمال الدلتا الرئيسي:
├── صناعي (30%) - أساسي
├── تجاري (20%)
├── سياحي (15%)
├── طبي (10%)
├── زراعي (10%)
├── إداري (10%)
└── منزلي (5%)

مخزن الإسكندرية الرئيسي:
├── سياحي (40% - أساسي)
├── تجاري (35% - أساسي)
└── صناعي (25%)

مخزن محرم بك شرق:
├── سياحي (70% - أساسي)
└── تجاري (30%)
```

## 🔍 **الاستعلامات المتاحة**

### **1. عرض شجرة المخازن الكاملة:**
```sql
SELECT * FROM get_warehouse_tree();
```

### **2. عرض المخازن الفرعية لمخزن معين:**
```sql
SELECT * FROM get_child_warehouses(1); -- معرف المخزن
```

### **3. عرض قطاعات مخزن معين:**
```sql
SELECT * FROM get_warehouse_sectors(2); -- معرف المخزن
```

### **4. عرض المخازن حسب المستوى:**
```sql
-- المخازن الرئيسية
SELECT * FROM warehouses WHERE warehouse_level = 'country';

-- مخازن المدن
SELECT * FROM warehouses WHERE warehouse_level = 'city';

-- مخازن المناطق
SELECT * FROM warehouses WHERE warehouse_level = 'district';
```

## 🛠️ **الوظائف المتاحة في الواجهة**

### **في تبويب "شجرة المخازن":**
- ✅ **عرض الشجرة الهرمية** مع إمكانية التوسيع والطي
- ✅ **إضافة مخزن جديد** (رئيسي أو فرعي)
- ✅ **تعديل مخزن موجود**
- ✅ **حذف مخزن** (مع التحقق من المخازن الفرعية)
- ✅ **عرض تفاصيل المخزن**

### **في تبويب "إدارة القطاعات":**
- ✅ **عرض القطاعات المرتبطة** بالمخزن المختار
- ✅ **إضافة قطاع جديد** للمخزن
- ✅ **تعديل نسبة السعة** لكل قطاع
- ✅ **تحديد القطاع الأساسي**
- ✅ **إزالة قطاع** من المخزن

### **في تبويب "الإعدادات":**
- ✅ **عرض إحصائيات النظام**
- ✅ **معلومات المستويات الهرمية**
- ✅ **قائمة القطاعات المدعومة**

## 🔧 **استكشاف الأخطاء**

### **مشاكل شائعة وحلولها:**

#### **1. خطأ في ربط القطاعات:**
```sql
-- تحقق من وجود القطاعات
SELECT * FROM waste_sectors WHERE is_active = TRUE;
```

#### **2. مشكلة في الشجرة الهرمية:**
```sql
-- إعادة تعيين مسار هرمي
UPDATE warehouses SET hierarchy_path = id::text WHERE parent_warehouse_id IS NULL;
```

#### **3. خطأ في الصلاحيات:**
```sql
-- فحص سياسات RLS
SELECT * FROM pg_policies WHERE tablename = 'warehouses';
```
