# قائمة الملفات التي يجب نقلها إلى المجلد المحول

## ملفات صفحات التطبيق (app)

نعم، يجب نقل جميع ملفات `page.tsx` التي تم تحويلها من المجلد `src/app` إلى المجلد المحول `src-restructured/app` مع الحفاظ على نفس البنية الداخلية للمجلدات:

1. **ملفات الصفحات المحولة**:
   - `src/app/map/page.tsx` → `src-restructured/app/map/page.tsx`
   - `src/app/agents/page.tsx` → `src-restructured/app/agents/page.tsx`
   - `src/app/orders/page.tsx` → `src-restructured/app/orders/page.tsx`
   - `src/app/order-tracking/[id]/page.tsx` → `src-restructured/app/order-tracking/[id]/page.tsx`
   - `src/app/trips/page.tsx` → `src-restructured/app/trips/page.tsx`
   - `src/app/analytics/page.tsx` → `src-restructured/app/analytics/page.tsx`
   - `src/app/support/page.tsx` → `src-restructured/app/support/page.tsx`
   - `src/app/settings/page.tsx` → `src-restructured/app/settings/page.tsx`
   - `src/app/messages/page.tsx` → `src-restructured/app/messages/page.tsx`

2. **ملفات إضافية ضرورية**:
   - `src/app/layout.tsx` → `src-restructured/app/layout.tsx`
   - `src/app/providers.tsx` → `src-restructured/app/providers.tsx`
   - `src/app/globals.css` → `src-restructured/app/globals.css`
   - `src/app/page.tsx` → `src-restructured/app/page.tsx`

## كيفية النقل

1. **إنشاء البنية الداخلية أولاً**:
   يجب التأكد من وجود جميع المجلدات الفرعية قبل نسخ الملفات. مثلاً:
   ```
   mkdir -p src-restructured\app\map
   mkdir -p src-restructured\app\agents
   mkdir -p src-restructured\app\orders
   mkdir -p src-restructured\app\order-tracking\[id]
   mkdir -p src-restructured\app\trips
   mkdir -p src-restructured\app\analytics
   mkdir -p src-restructured\app\support
   mkdir -p src-restructured\app\settings
   mkdir -p src-restructured\app\messages
   ```

2. **نسخ الملفات**:
   بعد إنشاء المجلدات، يمكن نسخ كل ملف على حده:
   ```
   copy src\app\map\page.tsx src-restructured\app\map\page.tsx
   copy src\app\agents\page.tsx src-restructured\app\agents\page.tsx
   ... إلخ
   ```

   أو نسخ محتويات المجلد بالكامل:
   ```
   xcopy /E /I src\app\* src-restructured\app\
   ```

## الملفات التي لا يجب نقلها

الملفات التالية يمكن نقلها إلى مجلد `src-legacy` لأنها أصبحت زائدة:

1. `src/app/order-tracking/[id]/tracking-client.tsx` → `src-legacy/order-tracking-client.tsx`
2. `src/app/trips/trips-client.tsx` → `src-legacy/trips-client.tsx`

## ملاحظات هامة

1. **التأكد من استيرادات المسارات**: قد تحتاج لتعديل مسارات الاستيراد في الملفات المنقولة إذا كانت تشير إلى ملفات أخرى.

2. **الحفاظ على الإصدارات الأصلية**: من المهم عدم حذف الملفات الأصلية قبل التأكد من أن التطبيق يعمل بشكل صحيح.

3. **اختبار التطبيق**: بعد نقل الملفات، تأكد من تشغيل التطبيق واختبار جميع الصفحات للتأكد من عملها بشكل سليم.
