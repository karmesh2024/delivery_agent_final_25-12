# نظام الإدارة العليا للمخازن - دليل شامل

## 🏢 نظرة عامة على النظام الجديد

تم تطوير نظام إدارة المخازن الهرمي ليشمل **الإدارة العليا للمخازن** كأعلى مستوى في الهيكل الهرمي، يتبعها المخازن الرئيسية على مستوى الدولة، ثم مخازن المدن، وأخيراً مخازن المناطق الفرعية.

## 📊 الهيكل الهرمي الجديد

### 1. الإدارة العليا للمخازن (Admin Level)
- **المستوى**: الأعلى في الهيكل الهرمي
- **الوظيفة**: إدارة جميع المخازن الأخرى
- **المميزات**:
  - إعدادات خاصة بالنظام
  - هيكل وظيفي مخصص
  - إعدادات أمنية متقدمة
  - مراقبة عامة للنظام

### 2. المخازن الرئيسية (Country Level)
- **المستوى**: يتبع الإدارة العليا
- **الوظيفة**: إدارة المخازن على مستوى الدولة
- **المميزات**:
  - إدارة مخازن المدن التابعة
  - تنسيق العمليات على المستوى الوطني

### 3. مخازن المدن (City Level)
- **المستوى**: يتبع المخزن الرئيسي
- **الوظيفة**: إدارة المخازن على مستوى المدينة
- **المميزات**:
  - إدارة مخازن المناطق التابعة
  - تنسيق العمليات على المستوى المحلي

### 4. مخازن المناطق (District Level)
- **المستوى**: يتبع مخزن المدينة
- **الوظيفة**: إدارة المخازن على مستوى المنطقة
- **المميزات**:
  - إدارة المخزون المحلي
  - تنفيذ العمليات اليومية

## 📋 كيفية الاستخدام

### 1. إنشاء الإدارة العليا
1. انتقل إلى `/warehouse-management/hierarchy`
2. انقر على زر "إدارة عليا"
3. املأ بيانات الإدارة العليا
4. احفظ الإعدادات

### 2. إعدادات الإدارة العليا
1. انتقل إلى `/warehouse-management/admin-settings`
2. قم بتعديل الإعدادات العامة
3. أضف الأقسام في الهيكل الوظيفي
4. حدد إعدادات النظام والأمان
5. احفظ جميع التغييرات

### 3. إدارة الهيكل الهرمي
1. انتقل إلى `/warehouse-management/hierarchy`
2. استخدم زر "إضافة مخزن رئيسي" لإنشاء مخزن رئيسي
3. أضف مخازن المدن التابعة للمخزن الرئيسي
4. أضف مخازن المناطق التابعة لمخزن المدينة
