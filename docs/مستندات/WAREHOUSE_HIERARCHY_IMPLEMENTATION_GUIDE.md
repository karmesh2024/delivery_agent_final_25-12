# 🏗️ دليل تطبيق النظام الهرمي للمخازن

## 📋 نظرة عامة

تم تطوير نظام هرمي متكامل لإدارة المخازن مع الصلاحيات المتدرجة والتفويض الذكي. هذا النظام يسمح للإدارة العليا للمخازن بإنشاء وإدارة مخازن على مستويات مختلفة مع تفويض الصلاحيات بشكل متدرج.

## 🎯 المميزات الرئيسية

### 1. **الهيكل الهرمي للمخازن**
- **الإدارة العليا للمخازن** (المستوى الأعلى)
- **مخازن الدولة الرئيسية** (المستوى الثاني)
- **مخازن المدن** (المستوى الثالث)
- **مخازن المناطق** (المستوى الرابع)

### 2. **نظام الصلاحيات المتدرجة**
- صلاحيات مختلفة لكل مستوى
- تفويض ذكي للصلاحيات
- مراقبة وإشراف شامل

### 3. **واجهة إدارة شاملة**
- شجرة مرئية للمخازن
- إدارة الصلاحيات والتفويض
- تقارير مفصلة

## 🔧 الاستخدام

### إنشاء مخزن جديد

1. **من النظام الهرمي:**
   - انتقل إلى `/warehouse-management/hierarchy`
   - اضغط على "إضافة مخزن جديد"
   - اختر المستوى المناسب
   - حدد المخزن الأب (إذا لزم الأمر)

2. **من إعدادات الإدارة العليا:**
   - انتقل إلى `/warehouse-management/admin-settings`
   - اختر تبويب "الهيكل الهرمي للمخازن"
   - اضغط على "إضافة مخزن رئيسي"

### إدارة الصلاحيات

1. **تعديل صلاحيات مخزن:**
   - من النظام الهرمي، اضغط على أيقونة الدرع
   - اختر الصلاحيات المطلوبة
   - احفظ التغييرات

2. **تفويض صلاحيات:**
   - انتقل إلى تبويب "التفويض"
   - اضغط على "تفويض جديد"
   - اختر المخزن المفوض والمستفيد
   - حدد أنواع الصلاحيات ومدة التفويض

### عرض التقارير

1. **التقارير الهرمية:**
   - انتقل إلى تبويب "التقارير"
   - عرض إحصائيات شاملة للنظام
   - مراقبة الصلاحيات والتفويض

## 🛡️ الأمان والصلاحيات

### سياسات الأمان (RLS)

تم تطبيق سياسات الأمان التالية:

```sql
-- سياسة للهيكل الهرمي
CREATE POLICY "warehouse_hierarchy_policy" ON warehouse_hierarchy
    FOR ALL USING (
        auth.uid() IN (
            SELECT user_id FROM warehouse_users 
            WHERE warehouse_id = warehouse_hierarchy.warehouse_id
        )
    );

-- سياسة للصلاحيات
CREATE POLICY "warehouse_permissions_policy" ON warehouse_permissions
    FOR ALL USING (
        auth.uid() IN (
            SELECT user_id FROM warehouse_users 
            WHERE warehouse_id = warehouse_permissions.warehouse_id
        )
    );
```
