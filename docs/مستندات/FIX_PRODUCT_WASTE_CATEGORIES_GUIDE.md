## 🔧 خطوات التطبيق

### 1. **إعادة تشغيل التطبيق**:
```bash
npm run dev
```

### 2. **اختبار النظام**:
- انتقل إلى `/warehouse-management/admin-settings/sectors`
- انتقل إلى تبويب "فئات المنتجات"
- انتقل إلى تبويب "فئات المخلفات"
- يجب أن تظهر البيانات الافتراضية

### 3. **إضافة فئات جديدة**:
- يمكن إضافة فئات منتجات جديدة
- يمكن إضافة فئات مخلفات جديدة
- يمكن ربطها بالقطاعات المناسبة
