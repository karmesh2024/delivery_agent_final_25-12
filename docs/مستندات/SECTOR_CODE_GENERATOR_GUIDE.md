# دليل استخدام مولد كود القطاع

## 🎯 الميزة الجديدة: زر توليد كود القطاع

### 📍 الموقع:
- **الصفحة**: `/warehouse-management/admin-settings/sectors`
- **التبويب**: "القطاعات" (Sectors)
- **القسم**: "إضافة قطاع جديد"

## 🔧 كيفية الاستخدام:

### 1. **إدخال اسم القطاع**
```
مثال: "القطاع الصناعي"
```

### 2. **النقر على زر "توليد"**
- زر أزرق مع أيقونة 🔄
- يقع بجانب حقل "كود القطاع"
- نص الزر: "توليد"

### 3. **النتيجة التلقائية**
```
اسم القطاع: "القطاع الصناعي"
الكود المولد: "القط"
```

## 🧠 خوارزمية التوليد الذكية:

### **الحالة الأولى: كلمة واحدة**
```
الاسم: "صناعي"
الكود: "صنع"
```

### **الحالة الثانية: كلمتان**
```
الاسم: "القطاع الصناعي"
الكود: "القط"
```

### **الحالة الثالثة: أكثر من كلمتين**
```
الاسم: "القطاع الصناعي المتقدم"
الكود: "القط"
```

## ✨ المميزات:

### 1. **توليد ذكي**
- ✅ يدعم النصوص العربية
- ✅ يتعامل مع المسافات والأحرف الخاصة
- ✅ ينتج أكواد قصيرة وواضحة

### 2. **مرونة في الاستخدام**
- ✅ يمكن التوليد التلقائي
- ✅ يمكن الإدخال اليدوي
- ✅ يمكن التعديل بعد التوليد

### 3. **واجهة سهلة**
- ✅ زر واضح مع أيقونة
- ✅ رسائل تأكيد
- ✅ تلميحات مفيدة

## 🔄 خطوات الاستخدام:

### **الطريقة الأولى: التوليد التلقائي**
1. أدخل اسم القطاع
2. اضغط على زر "توليد"
3. راجع الكود المولد
4. عدل إذا لزم الأمر
5. احفظ القطاع

### **الطريقة الثانية: الإدخال اليدوي**
1. أدخل اسم القطاع
2. أدخل الكود يدوياً
3. احفظ القطاع

## 🎯 نصائح للاستخدام:

### **لأفضل النتائج:**
- ✅ استخدم أسماء واضحة ومختصرة
- ✅ تجنب الأحرف الخاصة في الاسم
- ✅ راجع الكود قبل الحفظ
- ✅ تأكد من عدم تكرار الكود

### **أمثلة جيدة:**
```
✅ "القطاع الصناعي" → "القط"
✅ "القطاع التجاري" → "القط"
✅ "القطاع الزراعي" → "القط"
```

### **أمثلة تحتاج مراجعة:**
```
⚠️ "القطاع الصناعي المتقدم" → "القط" (قد يحتاج تعديل)
⚠️ "القطاع التجاري الدولي" → "القط" (قد يحتاج تعديل)
```

- ✅ **سهولة في التذكر** والاستخدام
- ✅ **تنظيم أفضل** للقطاعات
