# 📋 خطة تنفيذ منظومة البحث الذكي (Zoon Search Implementation Plan) - الإصدار المطور 2.1

## المرحلة الأولى: إعداد البيئة والاتصال الهجين (Core & Infrastructure)
1.  **إعداد خادم SearXNG الخاص**: لضمان السيادة والخصوصية.
2.  **نظام Fallback ذكي ومتوافق (Serverless Ready)**:
    - التسلسل: (SearXNG -> DDG -> Firecrawl/External Service).
    - **تنبيه البرودكشن ⚠️**: بما أن النظام مستضاف على Vercel، تم استبدال Playwright المحلي بـ `@sparticuz/chromium` أو الاعتماد على خدمات خارجية مثل `Firecrawl` لتجنب مشاكل توافق الـ Serverless.
    - ضبط timeout حازم (5 ثوانٍ) لكل محرك لضمان سلاسة التجربة.
3.  **الانهيار اللبق (Graceful Degradation) 🛡️**: في حال فشل جميع المحركات، يرد الوكيل بصدق: *"لا أستطيع الوصول للإنترنت حالياً، سأجيبك بناءً على معرفتي المخزنة حتى عام 2024"*.

## المرحلة 1.5: المنسق الذكي (Smart Orchestration)
الهدف: توفير 60% من الموارد عبر كشف النية.
1.  **تفعيل `toolChoice: 'auto'`**: استخدام Vercel AI SDK للسماح للنموذج بالتقرير الذكي.
2.  **هندسة أوصاف المهارات (Decision-Oriented Descriptions)**: 
    - مثال: *"استخدم هذه الأداة فقط للأخبار الجارية أو معلومات ما بعد 2024؛ لا تستخدمها للبيانات الداخلية"*.

## المرحلة الثانية: هندسة البيانات (The Butcher Reranker)
الهدف: جودة Perplexity بتكلفة ضئيلة.
1.  **استراتيجية (1 Full + N Snippets)**: جلب المحتوى الكامل لأفضل نتيجة فقط، والاكتفاء بالمقتطفات (Snippets) للبقية لتوفير الـ Tokens.
2.  **خوارزمية الترجيح الهيكلي**: ترتيب النتائج بناءً على (طول الوصف، موثوقية النطاق، وتطابق العنوان).
3.  **تنظيف الـ Markdown**: استخدام `turndown` بصرامة لإزالة أي ضجيج قبل إرسالها للنموذج.

## المرحلة الثالثة: الأداء والأمن (Scale & Security)
1.  **نظام Redis Caching المطوّر**: مفتاح الكاش يعتمد على `Query + Category`.
2.  **حماية الموارد (Rate Limiting per User)**: تطبيق حد أقصى للبحث (مثلاً 50 طلب/ساعة) لحماية الخادم.
3.  **إدارة الطلبات المكثفة**: دمج Proxies داخل SearXNG.

## المرحلة الرابعة: تجربة المستخدم (Morphic Experience)
1.  **الـ Initial Tool Result**: عرض الروابط والبطاقات فوراً (تأثير Perplexity).
2.  **الرؤية البصرية**: جلب الصور المصغرة (Thumbnails) والأيقونات لتعزيز ثقة المستخدم.
3.  **دعم الـ Markdown الشامل**: عرض الجداول والقوائم البحثية بجمالية فائقة في `ZoonChat`.

## 📊 التقرير الختامي للتنفيذ (Project Status: 100% COMPLETE ✅)

### ✅ ما تم إنجازه بالكامل (The Grand Completion)
1.  **البنية التحتية والسرعة الفائقة**:

4.  **المراقبة والتحليلات (Observability)**:

5.  **نظام الاكتشاف الاستباقي (Proactive AI)**:
    - **Pulse Runner**: محرك يبادر بالبحث والتحليل وتوليد الرؤى الصباحية (Pulse API).
